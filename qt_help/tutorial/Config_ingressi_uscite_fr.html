Configuration des entrées/sorties

Dans les appareils à E/S bidirectionnelles, à entrées ou sorties analogiques, à E/S Fast ou à encodeurs, il est nécessaire d'indiquer au produit comment ces lignes doivent être utilisées dans son application et c'est pourquoi nous parlons de “configuration”.

Il est important que la configuration ait lieu avant leur utilisation, sinon la ligne sera bloquée pour l'autoprotection.

Les configurations des lignes numériques et analogiques s'effectuent dans le programme dans la “Task0_Init” du logiciel plc.

Voir le chapitre “Configuration des entrées/sorties TPAC1006, TPAC1007 et TPAC1008_02” pour la configuration des variables présentes sur les modèles TPAC1006, TPAC1007 et TPAC1008_02.

Voir le chapitre “Configuration des entrées/sorties TPAC1008_03” pour la configuration des variables présentes sur les modèles TPAC1008_03.

Voir le chapitre “Configuration des entrées/sorties TPLC050 et TPLC100” pour configurer les variables présentes sur les modèles TPLC050 et TPLC100.
