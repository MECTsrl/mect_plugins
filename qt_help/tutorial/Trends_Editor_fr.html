Editeur de Trends

Avec l' "éditeur de Trends", vous pouvez configurer les variables historisées (variables qui ont les paramètres S, F, V et X dans le champ “Update” de l“Editeur de Crosstable ”) dont vous voulez avoir le graphique en fonction du temps.

Dans l'“Editeur de Trends” vous pouvez configurer 4 graphiques.

ATTENTION: Pour pouvoir afficher les tendances, les variables de référence doivent être historisées (voir chapitre “Fonctionnement de l'historique”) et doivent avoir été activées par la fonction logStart().
