Personalizar el fondo de una página

Puede personalizar el fondo de una página cambiando su color, añadiendo una imagen y colocando un borde.

Para ello, proceda del siguiente modo:

  1. Pulse el botón derecho del ratón en un punto libre de la página

  2. Seleccione la opción “Change styleSheet”

  3. Aparecerá la siguiente imagen:

  1. Coloque el cursor dentro de los corchetes y seleccione en el caso de:

    • una imagen: “Add Resource → image”

    • un color: “Add Color → background-color”

NOTA: Se recomienda encarecidamente no utilizar el degradado para establecer el fondo de una página porque el dispositivo sería lento. Utilice un solo color o una imagen para superar esta situación.


Personalizar la Splash screen

Para personalizar la Splash screen, simplemente sobrescriba el archivo “splash.png” en la carpeta “config” del proyecto. La imagen de la splash debe estar en formato png, de la misma resolución que la pantalla del producto utilizado:

  • TPAC1006/TP1057 => 320x240;

  • TPAC1007/TP1043 => 480x272;

  • TPAC1008/TP1070 => 800x480.
