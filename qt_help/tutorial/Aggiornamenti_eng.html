Update

Before upgrade read the “changelog.txt” file on Mect website in “DOWNLOAD Mect Suite” area.

To update, download the files from the mect.it site, specifically: Software and Services => Mect Suite => DOWNLOAD Mect Suite => Stable release 3.x.x => Mect Suite 3.x.x => Update Device

WARNING: DO NOT SWITCH OFF THE SYSTEM DURING THE UPGRADE.

Update for device:

  • img_sysupdate-3.x-ProductName.ext2 and sysupdate_img_3.x.sh:

    • By USB: copy files “img_sysupdate-3.x-ProductName.ext2” and “sysupdate_img_3.x.sh” on an USB key and restart the device with USB key inserted.

      For example, with TPAC1007_04_AC device:

      • img_sysupdate-3.0.0-TPAC1007_04_AC.ext2

      • sysupdate_img_3.0.0.sh

    The update of 3.x version takes about 2 minutes.

    Then switch off device, remove the USB pen drive and restart it. For TPAC1007 and TP1043 models use the micro-USB (type A) - USB adapter.

Devices without monitor: wait about 4 minutes before switching off the device.

WARNING: from 3.0.0 version it’s not possible to upgrade by Browser.
