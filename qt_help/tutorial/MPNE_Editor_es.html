Editor de MPNC

El configurador de módulos remotos es una herramienta gráfica útil para definir la configuración del sistema de automatización y, una vez definido el modelo de PLC o Panel de Operador Mect, permite la configuración de los nodos remotos “MPNE” y sus variables útiles para la aplicación.

El editor gráfico de los nodos remotos MPNE sólo es visible si se ha insertado al menos un módulo de la familia MPNE en el proyecto.


Para insertar un módulo, pulse el botón derecho del ratón en una fila de la Crosstable y seleccione “Paste MPNE1001 Module”.


En la siguiente ventana se puede definir la dirección del nodo que se está utilizando. Si el dispositivo Mect tiene varios puertos RTU disponibles, el sistema le permite asociar el puerto al nodo. Estos ajustes también se pueden modificar retroactivamente en el editor gráfico.


La interfaz es la siguiente:


En el campo “Model” aparece el código del PLC o del Panel de operador Mect.

En el campo “MPNE10” se puede seleccionar el nodo a utilizar para la configuración del sistema. La opción será múltiple si el mismo PLC o panel de operador Mect se combina con varios nodos remotos.

El campo “Port” se habilita cuando el modelo de PLC o Panel de Operador Mect tiene más de un puerto serie 485 y permite elegir el puerto con el que conectar el nodo remoto.

Il campo “Node Id” es la dirección del nodo remoto y puede ser definido por el usuario.

El botón cuadrado permite renombrar las variables asociadas a los módulos cambiando el prefijo “MPNE_” asignado por defecto por el sistema y el mismo para todos los módulos.

El campo “Exp1” define la configuración del módulo de expansión n°1 entre: “no Module, 8 Digital Input, 8 Digital Output, 2 AI – 1 AO”.

El campo “Exp2” define la configuración del módulo de expansión n°1 entre: “no Module, 8 Digital Input, 8 Digital Output, 2 AI – 1 AO”.

Nota: En el editor gráfico, los módulos de salida con 4 u 8 relés del módulo MPNE no están incluidos en las opciones, ya que equivalen a la expansión "8 Digital Output".


El botón de tres estados permite filtrar las variables de la lista y se compone de tres partes: una gris oscura, una gris clara y una azul. Dependiendo del estado, se visualizan diferentes variables:

  • El botón sólo con el gris oscuro activado muestra las variables de la tarjeta ÚNICAMENTE.

  • El botón con los colores gris oscuro y gris claro activado muestra las variables de la tarjeta y las variables de los demás módulos de expansión utilizados en el sistema.

  • El botón con el gris oscuro, gris claro y azul claro activado muestra las variables de la tarjeta, las variables de los otros módulos de expansión en uso y las variables aún no asociadas a ninguna expansión, pero que podrían ser utilizadas en el sistema.


Cuando se inserta un nuevo módulo, las variables de los módulos insertados se activan automáticamente y se muestran en la siguiente tabla. Para ver las variables asociadas a los otros módulos, utilice el botón three-state .

Haciendo clic en uno de los módulos insertados en el sistema, sólo se muestran en la tabla las variables del módulo seleccionado.

La tabla está en modo READ-ONLY. Haciendo clic en la variable deseada se puede acceder directamente al editor de Crosstable donde se pueden modificar los datos.
