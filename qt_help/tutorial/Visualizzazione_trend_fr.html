Affichage d'un trend

Insérez un ATCMbutton et mettez dans la propriété page_Name (à droite) le nom de la page de trend, trend1 par défaut.

Le graphique fonctionne en deux modes:

  • online: Le graphique défile automatiquement pour que les derniers échantillons soient toujours visibles;

  • offline: le graphique est statique et même si de nouveaux échantillons sont disponibles pour être affichés, ils ne seront affichés que s'ils sont contenus dans la zone actuellement affichée.

Au début, vous verrez les morceaux qui composent la famille en temps réel (on-line).

Les familles de trend sont composées de jusqu'à 4 graphiques. Voici la description des boutons à droite:

sauvegarde sur le panneau la capture d'écran du graphique courant

    • Appuyez sur les touches gauche et droite pour désactiver le mode on line.

    • Haut et bas ne seront activés que s'il y a des valeurs des variables en dehors de la plage affichée.

En bas à gauche, une étiquette affichera la valeur de la variable à laquelle le trend se réfère.

Sur la gauche du graphique se trouve un bouton qui vous permet de sélectionner le graphique que vous voulez avoir au premier plan (la couleur change en fonction du graphique choisi). Automatiquement, l'axe des valeurs de la variable courante sera affiché et sur le bouton en bas, la description du graphique courant sera affiché (ex : Entrée Analogique 1).

Pour modifier les réglages des pistes, vous pouvez utiliser l “Editeur de Trends ” (en faisant de nouveaux le deploy du projet depuis QtCreator) ou directement depuis l'appareil en appuyant sur le bouton en bas/au centre (dans le cas de la première image du chapitre, les mots “Analog Input 1”).

Pour chaque graphique, vous devez spécifier s'il est visible ou non, la variable à laquelle il se réfère, une description possible, la couleur que vous voulez associer et la plage de valeurs assumées par la variable (Ymin/Ymax).

Les graphique peuvent être visualisés avec les axes des temps soit en horizontal (landscape) soit en vertical (portrait).

Les boutons permettent d'éliminer le trend et/ou la variable à afficher dans le trend.

Une fois que les valeurs du graphique ont changé, appuyez simplement sur le bouton Save si vous voulez écraser la famille actuelle, Save as si vous vouliez créer une nouvelle famille de trend, qui sera composée du graphique avec les changements effectués plus les autres graphiques de la famille source, Save to USB si vous voulez enregistrer la famille de trend sur la clé USB. Le fichier enregistré sur la clé USB est similaire à celui enregistré à l'aide de l'éditeur de Trends.

En appuyant sur le bouton vous pouvez créer un nouveau trend, comme si vous utilisiez l'éditeur de trends.

ATTENTION: toutes les modifications effectuées directement sur l'appareil ne seront pas visibles depuis l'éditeur de Trends , sauf si vous sauvegardez sur USB et importez le nouveau trend depuis l'éditeur de Trends (à l'aide de la touche Import Trend).
