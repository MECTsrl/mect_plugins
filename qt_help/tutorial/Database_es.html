Acceso a bases de datos MySQL/SQLite

Para acceder a las bases de datos MySQL/SQLite, añada lo siguiente al archivo de proyecto (.pro)

   QT += sql
   QTPLUGIN += qsqlite

Introduzca el siguiente código, por ejemplo en automation.cpp (faltan las comprobaciones de errores):

	  #include <QtSql> 
	  #include <QSqlDatabase> 
	  #include <QDebug>
	  #include <QSqlQuery>	
	  QSqlDatabase db; //Database object

Dentro de setup()

	  db = QSqlDatabase::addDatabase("QSQLITE", "ConnectionName");
    db.setDatabaseName("/local/flash/root/test.sqlite");
    if (!db.open()) {
        qDebug() << QString("Error occurred opening the database. %1") .arg(db.databaseName());
        qDebug() << QString("%1.")  .arg(db.lastError().text());
    }
    else  {
        qDebug() << QString("Data base %1 Opened") .arg(db.databaseName());
    }

El primer parámetro de addDatabase dice qué driver usar (en este caso el driver SQLite), el segundo le da un nombre a la conexión, y se usa para abrir más de una DB al mismo tiempo.

Dentro del loop()

{
   QString testo = "SELECT * FROM TABELLA_X" ;
   QSqlQuery query(testo, db); 
   testo += "\n";
    if (!query.exec()) { 
       testo += QString("Open error [%1]").arg(db.lastError().text());
    } else { 
       for (int n = 0; n < query.record().count(); n++)
       { 
           testo += query.record().fieldName(n) + " - "; 
       } 
       testo += "\n";
	
       while(query.next()) { 
               for (int n = 0; n < query.record().count(); n++) 
               { 
                   testo += query.value(n).toString() + " | ";
	
               } 
               testo += "\n";
       } 
   }
   query.finish(); 
   // 
}

En el código debe cerrar el archivo, o mejor dicho, el código de apertura de la base de datos debe ejecutarse sólo cuando sea necesario y el archivo debe cerrarse inmediatamente después, en una secuencia como ésta.

// Closing DB 
    if (db.isOpen())
        db.close();

Además (para versiones menores de MectSuite 3.3.x), se debe agregar una carpeta “sqldrivers” al TPAC que contenga enlaces simbólicos a los plugins de interfaz de la DB:

	mkdir sqldrivers
	cd sqldrivers
	ln –s /usr/lib/libsq* ./
	ln –s /usr/lib/libqsqlite* ./
	cd
