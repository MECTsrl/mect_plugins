Cloner

"Cloner" est un utilitaire de récupération pour les panneaux Mect.

Procédure de démarrage de la récupération:

  • Copiez img_cloner_3.x.x.ext2, sysupdate_cloner_3.x.x.sh et _ysupdate_img_3.x.x.sh dans le dossier racine d'une clé USB (partition unique formatée FAT32 - MBR ms-dos).
    Attention: Aucun autre fichier "sysupdate_" ne doit se trouver sur la clé USB.

  • éteignez le dispositif.

  • Branchez la clé USB (Si le dispositif a le connecteur micro-USB, utilisez l'adaptateur micro-USB (type A) --> USB).

  • Allumez le dispositif.

Sur la clé, le dossier "cloner" sera créé et les images de sauvegarde y seront placées.


  • Sauvegarde l'application et les paramètres actuels. Le fichier de sauvegarde sera enregistré dans le dossier "cloner" sur la clé USB.

  • Restaure l'application et les paramètres. Le fichier de restauration doit se trouver dans le dossier "cloner" de la clé USB. Pour les fichiers rémanents, hmi.ini et log, il est possible de choisir de restaurer ("Restore"), de remettre à zèro ("Zero") ou d'ignorer ("Ignore") ces fichiers pendant la restauration.

  • Effectue la mise à jour de la Mect Suite. L'image de mise à jour (ex : img_sysupdate-3.x.x-TP1043_01_A.ext2) doit être insérée dans le dossier principal de la clé USB et doit être compatible avec le dispositif à mettre à jour. L'icône sera grisée s'il n'y a pas de mises á jour pour le panneau choisi.

  • Réinitialisation d'usine du panneau. Installez l'application "Simple" du panneau et réinitialisez les paramètres par défaut.

  • Menu VPN. (Désactivé)

  • Menu SSH. Permet de copier les clés SSH sur l'appareil. Les clés doivent être placées sur la clé USB dans le dossier "keys". si le dossier n'est pas présent vous devez le créer dans le dossier principal de la clé USB.

  • Menu de règlage de l'heure. (Désactivé)

  • Menu Réseau. Permet de modifier les paramètres wifi, ethernet et mobile.
