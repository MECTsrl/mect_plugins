Cloner

"Cloner" is a recovery utility for the Mect panels.

Recovery startup procedure:

  • Copy img_cloner_3.x.x.ext2, sysupdate_cloner_3.x.x.sh and _ysupdate_img_3.x.x.sh in the main directory of a USB pen drive (It must be a single partition FAT32 - ms-dos MBR).
    Warning: No others "sysupdate_" file must be on the USB drive.

  • Shutdown the device.

  • Plug in the USB drive (If the device have the micro-USB connector, use the micro-USB (type A) --> USB adapter).

  • Power on the device.

The backup images will be copied on the "cloner" directory into the pen drive.


  • It backups the application and current settings. The backup file will be saved on the "cloner" directory into the pen drive.

  • It restores the application and the settings. The restore file must be on the "cloner" directory of the pen drive. For retentive, hmi.ini and log files it is possible to choose if to restore ("Restore"), to reset ("Zero") or to ignore ("Ignore") these files during the restore.

  • It starts the Mect Suite update. The update image (ex: img_sysupdate-3.x.x-TP1043_01_A.ext2) must be inserted on the main directory of the USB drive and must be the aligned with the device model. The icon will be grey if no updates are available.

  • Factory Reset. Install the "Simple" application and reset the default settings.

  • VPN menu. (Disabled)

  • SSH menu. A menu for transfering or removing the SSH keys. The keys must be in the "keys" into the USB Drive. If the directory does not exist it can be created in the main directory of the USB Drive.

  • Time Set menu. (Disabled)

  • Network menu. Menu settings for wifi, ethernet and mobile.
