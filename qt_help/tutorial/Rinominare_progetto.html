Rinominare un progetto

Per rinominare la cartella di progetto creata con QtCreator occorre svolgere i seguenti passi:

  1. Accertarsi di aver chiuso il progetto che si vuole rinominare.

  2. Rinominare la cartella di progetto (non inserire caratteri speciali (escluso _ ) o spazi).

  3. Eliminare dalla cartella di progetto il file che ha estensione “.pro.user”.

  4. Rinominare il file con estensione “.pro” con il nome che si è scelto nel punto 2.

  5. Rinominare il file con estensione “.pro.shared” con il nome che si è scelto nel punto 2.

  6. Aprire il nuovo progetto da QtCreator.

  7. All’apertura apparirà la seguente immagine, premere il bottone “Configure Project”.

  1. Effettuare un “Run qmake” e un “Clean All”, vedi capitolo “Errori di compilazione”

  2. Fare il “Build” del progetto.

  3. Impostare i parametri per l’invio del progetto al dispositivo:

    • Disabilitare, sotto la voce “Upload files via SFTP”, il flag “Incremental deployment”;

    • Cliccare su “Add Deploy Step” e selezionare “Run custom remote command”

    • Scrivere “stop” (tutto minuscolo) e spostare il blocco “Run custom remote command” di una posizione verso l’alto, tramite la freccia evidenziata in giallo (premere Details).
