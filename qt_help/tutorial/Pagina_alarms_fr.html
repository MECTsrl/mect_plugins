Page alarms

Pour visualiser la page d'alarme, vous pouvez utiliser un ATCMbutton et entrer le mot “alarms” dans la propriété “pageName”. Voyez chapitre “Pages de librairie”.

En haut, il y a une liste déroulante dans laquelle vous pouvez sélectionner le type (All, Alarm e Event). Ces sélections seront utilisées pour les alarmes affichées dans la liste déroulante ci-dessous. Cet objet liste les alarmes qui sont actuellement visibles.

Les alarmes peuvent être affichées en 3 couleurs différentes:

  • Rouge: condition d'alarme toujours présente.

  • Bleu: l'opérateur a acquitté (ACK) mais la condition d'alarme est toujours présente.

  • Gris: condition d'alarme renvoyée. L'affichage de l'alarme disparaît après l'acquittement (ACK).

Immédiatement en dessous se trouve une bannière qui montre à son tour les différentes alarmes actuellement visibles.


En bas, à côté des boutons Home et Back habituels, il y a aussi les boutons suivants:

  1. bouton pour effectuer la reconnaissance d'une alarme préalablement sélectionnée dans la liste déroulante ci-dessus.

  2. bouton pour faire la reconnaissance de toutes les alarmes visibles.

  3. bouton pour désactiver le son de l'alarme.

  4. bouton pour aller à la page d'historique des alarmes.
