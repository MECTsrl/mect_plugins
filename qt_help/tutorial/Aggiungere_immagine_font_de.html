Bilder / Schriften hinzufügen

Damit ein Element außerhalb des Projekts (Bilder etc...) sichtbar und in das Projekt eingebunden werden kann, muss es der Ressourcendatei hinzugefügt werden. Führen Sie dazu die folgenden Schritte aus:

  • Fügen Sie Ihr eigenes Bild (Erweiterung .png) / Ihre eigene Schrift (Erweiterung .ttf) durch Datie Explorer, in den Ordner “icons” oder “fonts” " im Projektordner ein;

  • Doppelklicken Sie auf die .qrc-Datei im Abschnitt “Resources” im Projektstrukturfenster.

  • Drücken Sie die Taste “Add” um eine neue Ressource hinzuzufügen;

  • Speichern Sie die Datei resources.qrc.

An dieser Stelle kann das eingefügte Objekt innerhalb des Projekts gesehen werden.
