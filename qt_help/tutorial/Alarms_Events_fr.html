Alarms-Events

L'onglet "Alarms-Events" permet de visualiser toutes les variables d'alarme et d'événement.


L'interface est la suivante:

Cet écran affiche toutes les informations des variables définies comme alarme ou événement.

Le tableau est en mode READ-ONLY. En cliquant sur la variable désirée, vous pouvez accéder directement à l'éditeur de la Crosstable où vous pouvez modifier les données.
