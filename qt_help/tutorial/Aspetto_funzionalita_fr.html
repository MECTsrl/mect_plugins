Aspect et fonctionnalité

Lorsque vous sélectionnez l'élément, ses propriétés apparaissent dans la fenêtre inférieure droite de la fenêtre Qtcreator dans la section intitulée ATCMbutton.


En modifiant une ou plusieurs de ces propriétés, vous pouvez personnaliser l'aspect graphique et certaines fonctions. Pour effectuer la modification, sélectionnez la colonne de droite de la ligne correspondante.

En détail pour les fonctionnalités seront disponibles:

  • pageName: si vous voulez lier le bouton à un changement de page, vous devez entrer le nom de dla page de destination.

  • remember: au cas où vous voulez garder une trace de la page de destination pour pouvoir utiliser le mot- clé BACK (spécifié dans pageName).

  • passwordVar: si vous voulez protéger l'action du bouton avec un mot de passe numérique, vous devez écrire ici la variable de la Crosstable qui contient le mot de passe souhaité ou directement un mot de passe numérique statique.

  • statusvar: si vous voulez lier le bouton à un changement d'état d'une variable à bascule, vous devez entrer le nom de la variable de destination.

  • statusReleasedValue: la valeur que le statusVar prendra lorsque le bouton sera relâché.

  • statusPressedValue: la valeur que le statusVar prendra lorsque le bouton sera pressé.

  • viewStatus: s'il est coché, il aura une couleur rouge ou verte respectivement si le bouton est utilisé en mode bascule, selon que le réglage de la variable statusvar a échoué ou non et si la bordure est visible.

  • visibilityVar: si vous voulez lier la visibilité ou non de l'objet en fonction de l'état d'une variable, vous devez saisir le nom de la variable destination.

En détail pour l'aspect graphique sera disponible:

  • bgColor: détermine la couleur de fond de l'objet.

  • borderColor: détermine la couleur du bord de l'objet, uniquement si l'épaisseur du bord est supérieure à zéro.

  • fontColor: détermine la couleur du texte de l'objet.

  • bgPressColor: est similaire à bgColor, lorsque le bouton est pressé.

  • borderPressColor: est similaire à borderColor, lorsque le bouton est pressé.

  • fontPressColor: est similaire à fontColor, lorsque le bouton est pressé.

  • text: est le texte qui sera affiché sur le bouton.

  • pressText: est similaire à text, lorsque le bouton est pressé.

  • icon: est l'icône qui sera affichée sur le bouton. Appuyer sur le bouton “…” permet d'insérer les icônes du système. La fenêtre suivante apparaîtra, choisissez l'icône et appuyez sur OK:

  • pressIcon: : il est similaire à icon, quand le bouton est pressé.

  • borderWidth: est l'épaisseur en pixels du bord de l'objet.

  • borderRadius: est le rayon de courbure des bords de l'objet.

  • apparence: c'est le style avec lequel vous voulez montrer l'objet.

NOTE: dans toutes les propriétés, à l'exception des ATCMplugins, il est possible d'afficher le “Context Help” de Qt pour avoir une description du fonctionnement: sélectionnez la ligne souhaitée et appuyez sur la touche “F1” du clavier.

Pour utiliser les propriétés de l'objet par code, tapez dans Help→Index: QPushButton
