System variables

In the following table there are the main system variables. These variables are present in the Crosstable and are visible starting from row 5300 of “Crosstable Editor”. For example, if in HMI program you want to command an ATCMbutton to change the digital output 1 or to initialize in PLC digital output 1 to a value, use the variable PLC_DigOut_1.
