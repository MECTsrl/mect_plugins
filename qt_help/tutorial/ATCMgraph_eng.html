ATCMgraph

Object to show a 2 trace graph.

To use follows the steps:

  1. Drag object on page and double click on it.

  2. Define x1Variable with Crosstable variable name containing the value representing x value of trace 1. Press “…” button to select desired variable.

  3. Define x1Min and x1Max with Crosstable variable name containing the value representing x value interval of trace 1. If fixed write the number instead of variable, otherwise press “…” button to select desired variable. To give a flow effect leave default variables: PLC_timeMin and PLC_timeMax.

  4. Define y1Variable with Crosstable variable name containing the value representing y value of trace 1. Press “…” button to select desired variable.

  5. Define y1Min and y1Max with Crosstable variable name containing the value representing y value interval of trace 1. If fixed write the number instead of variable, otherwise press “…” button to select desired variable. Define x2Variable with Crosstable variable name containing the value representing x value of trace 2. Press “…” button to select desired variablea.

  6. Define x2Min and x2Max with Crosstable variable name containing the value representing x value interval of trace 2. If fixed write the number instead of variable, otherwise press “…” button to select desired variable. To give a flow effect leave default variables: PLC_timeMin e PLC_timeMax.

  7. Define y2Variable with Crosstable variable name containing the value representing y value of trace 2. Press “…” button to select desired variable.

  8. Define y2Min and y2Max with Crosstable variable name containing the value representing y value interval of trace 2. If fixed write the number instead of variable, otherwise press “…” button to select desired variable.

  9. Define Display1 and Display2 with Crosstable variable name containing trigger values to show trace 1 and trace 2. If empty, triggers are on input value change. Press “…” button to select desired variable.
