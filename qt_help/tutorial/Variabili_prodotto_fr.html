Variables de produit (E/S locales)

A l'intérieur de la Crosstable se trouvent les variables de produit les plus importantes. Plus précisément ces variables sont visibles à partir de la ligne 5300 de “Crosstable Editor”.

Par exemple, si vous voulez contrôler à partir de l'IHM l'action d'un bouton ATCM qui avec la pression change la sortie numérique 1 ou si vous voulez initialiser à partir de PLC, la sortie numérique 1 à une valeur, vous devez utiliser directement la variable PLC_DigOut_1.

Voir le chapitre “Modèles TPAC1008_02” pour voir la liste des variables présentes sur les modèles TPAC1008_02.

Voir le chapitre “Modèles TPAC1007” pour voir la liste des variables présentes sur les modèles TPAC1007.

Voir le chapitre “Modèles TPAC10008_03” pour voir la liste des variables présentes sur les modèles TPAC10008_03.

Voir le chapitre “Modèles TPLC050 / TPLC100” pour voir la liste des variables présentes sur les modèles TPLC050 et TPLC100.
