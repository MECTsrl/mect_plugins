Impostazione delle password

Ci sono 2 modi differenti per visualizzare / impostare le password:

  • Usare la pagina di libreria dedicata “set_password”.

  • Usare la pagina di libreria generica “menu”.

Uso della pagina “set_password”:

Per visualizzare la pagina di impostazione delle password, usare un ATCMbutton e inserire il nome della pagina di libreria “set_password” nella Property “pageName”.

Si consiglia di proteggere il cambio pagina del ATCMbutton con una password differente, statica o dinamica, inserendola nel campo “PasswordVar”.

Nella pagina visualizzata, si potranno impostare le password dei 3 livelli: “Admin”, “SuperUser” e “User”.


Uso della pagina “menu”:

Tutti i sistemi hanno come password di default “0” per tutti e tre i livelli previsti.

Poiché le password di default sono identiche per tutti e tre i livelli, la prima operazione di login effettuata dall’applicazione è gestita in modo gerarchico partendo dal livello a priorità più alta. Pertanto il primo login effettuato nell’applicazione sarà attribuito ad “Admin” e alla eventuale pressione del tasto ‘Password’ si modificherà la password di “Admin”.

Il successivo login con password “0” sarà attribuito all’utente “SuperUser” del quale, se desiderato, si potrà modificare la password. Il successivo login con password “0” sarà attribuito all’utente “User” del quale, se desiderato, si potrà modificare la password.

L’ultimo login con password “0” sarà attribuito ad “Operator”.
