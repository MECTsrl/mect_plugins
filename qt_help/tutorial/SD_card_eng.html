Micro SD card insertion

On panels TPAC1007_04 it is possible to insert a micro SD card up to 64 GB (FAT format and only partition). First time the following windows appears:

It is possible to choose 4 modes:

  • Remind me at next boot: this windows appears again next boot;

  • Ignore it: ignore and don’t show again and mount sd card;

  • Use it as main storage (HMI+PLC+LOG): card is used to store the HMI, PLC application with LOGS.

    WARNING: if card is removed, the application is not present on panel any more. If card is moved on another panel (same model), it starts with application in SD card.

  • Use it as data storages (only LOG): card is used only for LOGS.
