Look and functionality customization

Selecting the element, its property are shown in the lower right size of the window of QtCreator in the section ATCMlabel. Modifying one or more properties the graphic aspect and some functionalities can be modified. To modify them select right column of the corresponding row.

In particular for functionalities:

  • variable: defines Crosstable variable where is stored current value of the label.

  • prefix: prefix of the value of label.

  • suffix: suffix of the value of the label.

  • max: max possible value.

  • min: min possible value.

  • viewStatus: if checked the color is green or red if button is used as toggle, depending on setting of variable and if the edge is visible.

  • visibilityVar: if an object visibility must be linked to a variable value, write the name of the variable here.

  • format: format of the value to display: decimal, hexadecimal or binary.

In particular for graphic aspect:

  • bgColor: background color of the object.

  • borderColor: color of the object, only if the edge is > 0.

  • fontColor: text color of the object.

  • bgSelectColor: like bgColor, when label is pressed to write value.

  • borderSelectColor: like borderColor, when label is pressed to write value.

  • fontSelectColor: like fontColor, when label is pressed to write value.

  • borderWidth: pixel thikness of the object.

  • borderRadius: curve radius of corner of the object.

  • apparence: style of the object.

WARNING: in all Properties, except ATCM plugin, it is possible to see the “Context Help” of Qt to see a description of the functionality: select the desired row and press “F1” key.

To use the code properties, to write in Help →Index: QPushButton.
