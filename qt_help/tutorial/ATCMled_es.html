ATCMled

Este objeto permite la visualización de una variable entera de la Crosstable en un estilo de leds: si la variable es cero, verá un led apagado, de lo contrario estará encendido.

Para utilizarlo, siga los pasos siguientes:

  1. Arrastre el objeto a la página.

  2. Haga doble clic en el objeto.

  3. Defina el campo variable con el nombre de la variable Crosstable de la que desea leer/escribir. Pulse el botón “…” para seleccionar la variable deseada.

  4. Defina la variable visibilityVar con el nombre de la variable Crosstable de la que se desea leer el valor a utilizar para mostrar o no el objeto. Pulse el botón “…” para seleccionar la variable deseada.
