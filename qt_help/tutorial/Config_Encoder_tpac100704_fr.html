Configuration encodeur

TPAC1008 / TPLC100: Avant d'effectuer la configuration, attendez que la variable “CH0_01_NODERUN” soit “TRUE”.

TPAC1007 / TPLC050: La configuration peut être effectuée dans la “Task0_Init” du logiciel plc.


Variable

Configuration

PLC_EncoderStart

  • 0 encodeur désactivé

  • 1 encodeur bidirectionnel activé

  • 2 compteur

Ex°: Encodeur = bidirectionnel

PLC_EncoderStart := 1;
