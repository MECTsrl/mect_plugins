La clase NTP Client

La clase ntpclient es una librería para la sincronización con el protocolo NTP que puede ser incluida en sus proyectos.

Las funciones que se pueden utilizar son:

  1. QString getNtpServer();
  2. int getTimeout_s();
  3. int getOffset_h();
  4. int getPeriod_h();
  5. void setNtpParams(const QString &ntpServer, int ntpTimeout_s, int ntpOffset_h, int ntpPeriod_h);
  6. void requestNTPSync();
  7. void requestDateTimeChange(QDateTime newTime);

En el proyecto de ejemplo time_set_AnyTPAC043 en mect_apps/Examples, todas estas funciones se utilizan para recrear la página del sistema Time Set.
