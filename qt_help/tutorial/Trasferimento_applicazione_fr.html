Transfert de l'application vers l'appareil

Pour transférer votre application du PC vers le panneau, vous devez connecter le panneau au PC via un câble réseau ou vérifier que les deux sont connectés entre eux.

La première fois que vous voulez envoyer l'exécutable (Deploy) au panneau, sélectionnez “Projects” → “Run” et écrivez “stop” (en minuscules) dans “Run custom remote command” :

Il est conseillé de travailler en mode “Release” car l'application est moins “lourde” pour la cible.

Si nécessaire, vous pouvez spécifier une adresse IP différent pour la cible dans “Tools → Options → Devices”.

Il est conseillé d'effectuer un test pour vérifier qu'il y a une connexion entre le PC et le panneau ; pour ce faire, appuyez sur le bouton “Test”.

La phrase de passe est : Vpn:Update1207

Une clé commune à tous les appareils Mect située en C:\Qt485\keys est utilisée pour déployer l'appareil ou y accéder.

Il est fortement recommandé de générer et d'utiliser une nouvelle clé pour vos propres appareils (Putty Installer: https://www.chiark.greenend.org.uk/~sgtatham/putty/latest.html).
