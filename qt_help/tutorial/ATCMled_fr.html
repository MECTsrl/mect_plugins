ATCMled

Cet objet permet d'afficher une variable entière de la Crosstable dans un style de led : si la variable est nulle, vous verrez une led éteinte, sinon elle sera allumée.

Pour l'utiliser, suivez les étapes ci-dessous:

  1. Faites glisser l'objet sur la page.

  2. Double-cliquez sur l'objet.

  3. Définissez le champ variable avec le nom de la variable Crosstable à partir de laquelle vous voulez lire. Appuyez sur le bouton “…” pour sélectionner la variable désirée.

  4. Définissez la variable visibilityVar avec le nom de la variable Crosstable à partir de laquelle vous voulez lire la valeur à utiliser pour afficher ou non l'objet. Appuyez sur le bouton “…” pour sélectionner la variable désirée.
