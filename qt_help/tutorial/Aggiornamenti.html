Aggiornamenti

Prima di effettuare l’aggiornamento leggere il file “changelog.txt” presente sul sito Mect nell’area “DOWNLOAD Mect Suite”.

Per effettuare l’aggiornamento scaricare i file dal sito mect.it, nello specifico: Software e servizi => Mect Suite => DOWNLOAD Mect Suite => Stable release 3.x.x => Mect Suite 3.x.x => Update Device

ATTENZIONE: NON SPEGNERE IL SISTEMA DURANTE L’AGGIORNAMENTO.

Aggiornamento del dispositivo:

  • img_sysupdate-3.x-NomeProdotto.ext2 e sysupdate_img_3.x.sh:

    • Via USB: copiare i file “img_sysupdate-3.x-NomeProdotto.ext2” e “sysupdate_img_3.x.sh” su una pen drive USB.

      Ad esempio, per il TPAC1007_04_AC si dovrà copiare sulla chiavetta i file:

      • img_sysupdate-3.x.x-TPAC1007_04_AC.ext2

      • sysupdate_img_3.x.x.sh

    Per la versione 3.x occorrono circa 2 minuti. Dopo aver visualizzato la scritta: “Please power off and remove the USB storage device” è possibile spegnere il dispositivo, togliere la chiavetta USB e riavviarlo. Per i modelli TPAC1007 e TP1043 serve l’adattatore micro-USB (tipo A) - USB.

Dispositivi senza monitor: attendere circa 4 minuti prima di spegnere il dispositivo.

ATTENZIONE: dalla versione 3.0.0 non è possibile fare l’aggiornamento via Browser.
