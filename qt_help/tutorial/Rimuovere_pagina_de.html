Eine Seite entfernen

Um eine bestehende Seite zu entfernen, gehen Sie in den Abschnitt “Form” aus dem Projektbaum auf der linken Seite und klicken Sie mit der rechten Maustaste auf die .ui-Datei, die sich auf die zu löschende Seite bezieht, und wählen Sie “Remove file”. Wählen Sie im nächsten Fenster “Delete file permanently” und drücken Sie “Close”.

Entfernen Sie die entsprechenden Seiten .cpp und .h

Achtung: Nach dem Entfernen der gewünschten Seite müssen Sie das Projekt “bereinigen” um Fehler bei der Kompilierung zu vermeiden.

Drücken Sie insbesondere Build → Clean All

Warten Sie, bis der Balken grün wird.

Dann drücken Sie Build → Run qmake

Warten Sie, bis der Balken grün wird.

Achtung: Führen Sie einen weiteren Build durch, falls ein Fehler gemeldet wird.
