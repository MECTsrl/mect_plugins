Páginas de librería

Dentro del proyecto hay páginas listas que se pueden utilizar; para ello es necesario escribir el nombre de la página en la propiedad “pageName” del ATCMbutton y específicamente:

  • Una ventana “display_settings” que permite la gestión del brillo y el ajuste del tiempo de espera del salvapantallas.

  • Una ventana “info” que implementa la visualización de la información de la placa y del software cargado.

  • Las ventanas “menu” y “options” que implementan las páginas básicas desde las que se muestran las demás páginas.

  • La ventana “recipe_select” abre la página de la receta preconstruida. La receta es la configuración en los dispositivos de campo de un determinado conjunto de parámetros con valores por defecto. La definición de una receta se realiza mediante la compilación de un archivo según los métodos indicados en el apartado Funcionamiento recetas.

  • La ventana “time_set” implementa la página para cambiar la hora y la fecha.Time set

  • Los archivos “trend” implementan la gestión y visualización de gráficos relacionados con los datos históricos. Ver el capítulo “Trends”.

  • La finestra di “store” " implementa la página para mostrar los historiales definidos. Dentro la Crosstable es posible declarar magnitudes como elementos a historizar.A partir de esta definición, cuando hay magnitudes a historizar, se genera un log diario en el que se graban los datos necesarios. Es posible definir uno o más subconjuntos de magnitudes historiadas. Ver el capítulo “Funcionamiento historial”.

  • La ventana “alarms” implementa la visualización del historial de alarmas. Ver el capítulo “Funcionamiento de las alarmas”.

  • La ventana “alarms_history” " implementa la visualización del historial de alarmas.

  • La ventana “data_manager” implementa una página de menú para entrar en las páginas de alarmas, historiales, tendencias y recetas.

  • La ventana “net_conf” abre la página de configuración de red: Ethernet, Wi-Fi y Mobile.

  • La ventana “system_ini” muestra todas las configuraciones del panel, desde el comportamiento gráfico, como la selección de la “HOME page” hasta la frecuencia de funcionamiento de los distintos buses. Reinicie el dispositivo después de cada cambio.

  • La ventana “login_pwd” muestra la página de inicio/cierre de sesión y cambio de contraseña. Ver el capítulo “Funcionamiento de las contraseñas”.

  • La ventana “set_password” muestra la página para configurar las contraseñas para los distintos niveles. Ver el capítulo “Funcionamiento de las contraseñas”.
