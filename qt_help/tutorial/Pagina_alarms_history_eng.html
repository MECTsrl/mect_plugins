Alarms_history page

To show history alarms page use an ATCMbutton and insert in the property “pageName” word “alarms_history”. See chapter “Library pages”.

In this page all previous alarms are shown:

Definition of RISE, ACK and FALL:

  • RISE: alarm signal.

  • ACK: acknowledgment of the alarm by the operator.

  • FALL: alarm returned.

It is possible to filter by day using the drop down menu. It is possible to download data on an USB pen with key active when USB key is inserted.
