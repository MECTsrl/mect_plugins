ATCMled

Dieses Objekt ermöglicht die Anzeige einer ganzzahlen Variablen des Crosstable in einem LED-Stil: Wenn die Variable Null ist, sehen Sie eine LED aus, sonst ist sie eingeschaltet.

Um es zu verwenden, führen Sie die folgenden Schritte aus:

  1. Ziehen Sie das Objekt auf die Seite.

  2. Doppelklicken Sie auf das Objekt.

  3. Definieren Sie das Feld variable mit dem Namen der Crosstable-Variablen, aus der Sie lesen/schreiben möchten. Drücken Sie die Taste “…”, um die gewünschte Variable auszuwählen.

  4. Definieren Sie die Variable visibilityVar mit dem Namen der Crosstable-Variablen, aus der Sie lesen/schreiben möchten, der zur Anzeige des Objekts verwendet werden soll oder nicht. Drücken Sie die Taste “…”, um die gewünschte Variable auszuwählen.
