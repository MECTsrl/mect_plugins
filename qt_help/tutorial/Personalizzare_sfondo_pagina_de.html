Anpassen des Hintergrunds einer Seite

Sie können den Hintergrund einer Seite anpassen, indem Sie ihre Farbe ändern, ein Bild hinzufügen und einen Rahmen platzieren.

Gehen Sie dazu wie folgt vor:

  1. Drücken Sie die rechte Maustaste an einem freien Punkt der Seite

  2. Wählen Sie das Element “Change styleSheet”

  3. Das folgende Bild wird angezeigt:

  1. Positionieren Sie den Cursor innerhalb der geschweiften Klammern und wählen Sie im Falle von:

    • ein Bild: “Add Resource → image”

    • eine Farbe: “Add Color → background-color”

HINWEIS: Es wird dringend davon abgeraten, den Farbverlauf zu verwenden, um den Hintergrund einer Seite einzustellen, da das Gerät langsam wäre. Verwenden Sie eine einzelne Farbe oder ein Bild, um diese Situation zu überwinden.


Anpassen des Splash-Screens

Um den Splash-Screen anzupassen, überschreiben Sie einfach die Datei “splash.png” im Ordner “config” des Projekts. Das Bild des Splash muss im png-Format sein, mit der gleichen Auflösung wie der Bildschirm des verwendeten Produkts:

  • TPAC1006/TP1057 => 320x240;

  • TPAC1007/TP1043 => 480x272;

  • TPAC1008/TP1070 => 800x480.
