Insertar una ventana emergente

Es posible asociar a la presión de un botón ATCM la visualización de una ventana emergente. Por ejemplo, esta ventana puede tener un icono de “información”, “aviso” o de mensaje “crítico”.

Siga el capítulo “Función implementada por el usuario” e introduzca el siguiente código:

QMessageBox::information(this,trUtf8(“USB error”), trUtf8(“Cannot mount the usb key”));

En este caso, al pulsar la tecla, aparecerá una ventana con el icono “información”

En el ejemplo, la escritura USB error corresponde al título, mientras que la escritura Cannot mount the usb key es el mensaje.

Introduzca el siguiente código al principio de pagexxx.cpp:

	  #include <QMessageBox>

Para visualizar el icono de mensaje “avviso” escriba:

QMessageBox::warning(this,trUtf8(“USB	error”), trUtf8(“Cannot mount the usb key”));

Para visualizar el icono de mensaje “crítico” escriba:

QMessageBox::critical(this,trUtf8(“USB	error”), trUtf8(“Cannot mount the usb key”));
