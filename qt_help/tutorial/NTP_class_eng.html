NTP Client Class

The ntpclient class is a library for synchronisation with the NTP protocol that can be included in your own projects.

The functions that can be used are:

  1. QString getNtpServer();
  2. int getTimeout_s();
  3. int getOffset_h();
  4. int getPeriod_h();
  5. void setNtpParams(const QString &ntpServer, int ntpTimeout_s, int ntpOffset_h, int ntpPeriod_h);
  6. void requestNTPSync();
  7. void requestDateTimeChange(QDateTime newTime);

In the example project time_set_AnyTPAC043 in mect_apps/Examples, all these functions are used to recreate the Time Set system page.
