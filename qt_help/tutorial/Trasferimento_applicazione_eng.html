Download to the panel of a graphic application

Connect operator panel to PC by a LAN cable.

First time you download the executable file (Deploy) you must select “Projects” → “Run” and write “stop” (lowercase) in “Run custom remote command” :

It's better to work in “Release” mode because the application is smaller for the target.

If necessary, it is possible to use a different IP address in the dialog in “Tools → Options → Devices”.

Please check connection between PC and panel. To do it press “Test” key.

The passphrase is: Vpn:Update1207

A common key for all Mect devices located in C:\Qt485\keys is used to deploy or access the device.

It is strongly recommended that you generate and use a new key for your own devices (Putty Installer: https://www.chiark.greenend.org.uk/~sgtatham/putty/latest.html).
