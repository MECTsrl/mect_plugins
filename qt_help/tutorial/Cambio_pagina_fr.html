Changer une page

Pour utiliser le bouton pour effectuer un changement de page, procédez comme suit:

  1. Faites glisser l'objet sur la page.

  2. Sélectionnez le bouton et mettez dans la propriété page_Name (à droite) le nom de la page à laquelle vous voulez aller (si vous voulez aller à la page d'accueil, écrivez simplement HOME, si vous voulez retourner à la dernière page visitée, écrivez BACK). Si vous souhaitez utiliser des pages système, consultez le paragraphe “Pages de librairie”.

  3. cochez la case de la propriété remember si vous voulez ajouter la page de destination à l'historique de la page pour pouvoir utiliser la fonction BACK plus tard.
