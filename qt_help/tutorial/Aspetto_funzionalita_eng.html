Look and functionality customization

If the element is selected, its property are shown in the right bottom side of QtCreator window in the ATCMbutton section, that can customize the look and some functionalities.


To change the value select the right side of the relevant row.

In particular for functionalities the fields are:

  • pageName: insert destination page name if button must change page.

  • remember: check to store page in order to use BACK keyword (specify in pageName).

  • passwordVar: to protect the button with password, write here the Crosstable variable containing the password or write directly a static password.

  • statusvar: to link the button to a toggle variable status change, write the destination variable name.

  • statusReleasedValue: value of statusVar when button is released.

  • statusPressedValue: value of statusVar when button is pressed.

  • viewStatus: if checked the color is green or red if button is used as toggle, depending on setting of variable statusvar sand if the edge is visible.

  • visibilityVar: if an object visibility must be linked to a variable value, write the name of the variable here.

In particular for graphic aspect is available:

  • bgColor: background color of the object.

  • borderColor: color of the object, only if the edge is > 0.

  • fontColor: text color of the object.

  • bgPressColor: like bgColor, when button is pressed.

  • borderPressColor: like borderColor, when button is pressed.

  • fontPressColor: like fontColor, when button is pressed.

  • text: text on button.

  • pressText: like text, when button is pressed.

  • icon: icon on button. With key “…” it is possible to insert system icons. The following windows appears, choose icon and press OK:

  • pressIcon: like icon, when button is pressed.

  • borderWidth: pixel thikness of the object.

  • borderRadius: curve radius of corner of the object.

  • apparence: style of the object.

WARNING: in all Properties, except ATCMplugin, it is possible to see the “Context Help” of Qt to see a description of the functionality: select the desired row and press “F1” key.

To use the code properties, to write in Help →Index: QPushButton.
