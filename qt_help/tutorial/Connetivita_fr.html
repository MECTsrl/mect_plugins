Connexion à distance (sMily)

Il est possible de connecter l'appareil à l'extérieur via le réseau LAN, Wi-Fi et Mobile. La connexion LAN est disponible directement à partir du connecteur Ethernet, tandis que les connexions Mobile et Wi-Fi sont disponibles via des périphériques USB dédiés.

Il est possible d'utiliser le service de connexion à distance sMily.

Grâce au service de connexion à distance sMily les opérations suivantes peuvent être effectuées, directement sur la machine du client:

  • La Mise à jour de la partie HMI;

  • La Mise à jour de la partie API;

  • Le débogage de l'application API (breakpoints);

  • Vous connecter au serveur Web de l'appareil;

  • Afficher et contrôler votre application via VNC.

Tout cela comme si l'appareil était sur votre bureau (si la vitesse de connexion le permet).

De plus, il est possible de monitorer les variables de votre application.

ATTENTION: seuls les modèles de clés Wi-Fi et Mobile (3G) fournis par Mect ont été testés et garantis.
