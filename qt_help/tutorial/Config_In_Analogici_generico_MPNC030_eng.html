Analog input configuration

The configuration can be performed within the “Task0_Init” of the plc software or when the Modbus connection is activate (RTUX_STATUS = 2).


Variable

Analog input configuration

XX_AnInConf_A..D

  • 0 not configured

  • 1 current

  • 2 voltage

  • 3 TCJ (thermocouple J)

  • 4 TCK (thermocouple K)

  • 5 TCT (thermocouple T)

  • 6 PT100E (Resolution 1 degree) range: -40 +800°C

  • 7 PT100R (Resolution 0.1 degree) range: -40 +200°C

  • 8 TCS (thermocouple S)

  • 9 TCB (thermocouple B)

  • 10 TCR (thermocouple R)

Ex°: ch1 = PT100E / ch2 = 0-10V / ch3 = TCJ / ch4 = TCK

XX_AnInConf_A := 16#4326;
