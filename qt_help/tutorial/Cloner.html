Cloner

Il "Cloner" è una utility di recovery per i panneli Mect.

Procedura per l'avvio della recovery:

  • Copiare i file img_cloner_3.x.x.ext2, sysupdate_cloner_3.x.x.sh e _ysupdate_img_3.x.x.sh nella cartella principale di una pen drive USB (formattata FAT32 a singola partizione - ms-dos MBR).
    Attenzione: NON devono essere presenti altri file di aggiornamento "sysupdate_"

  • Spegnere il dispositivo.

  • Inserire la chiavetta USB (se il dispositivo ha una micro-USB, utilizzare un adattatore micro-USB (tipo A) --> USB).

  • Accendere il dispositivo.

Sulla pen drive verrà creata la cartella "cloner" dove saranno inserite le immagini di BACKUP.


  • Effettua il backup dell'applicazione e delle impostazioni correnti (micro SD card esclusa). Il backup verrà salvato sulla penna USB nella cartella "cloner".

  • Effettua il restore dell'applicazione e delle impostazioni correnti (micro SD card esclusa). Il file di restore deve essere sulla penna USB nella cartella "cloner". Per ritentive, hmi.ini e logs è possibile scegliere se effettuare il restore ("Restore"), azzerare ("Zero") oppure lasciare quanto sul dispositivo ("Ignore")

  • Effettua l'update della Mect Suite. L'immagine di update (ex: img_sysupdate-3.x.x-TP1043_01_A.ext2) deve essere inserita nella cartella principale della penna USB e deve essere coerente con il dipositivo da aggiornare. L'icona sarà grigia se non sono presenti Update per il pannello scelto.

  • Factory Reset del pannello. Installa l'applicazione "Simple" del pannello e resetta le impostazioni.

  • Menù VPN. Installa/rimuove il certificato che consente una connessione privata di rete verso la piattaforma sMily. I certificati devono essere inseriti sulla penna USB nella cartella "ovpn", se non è presente la cartella è necessario crearla nella directory principale della penna USB.

  • Menù SSH. Permette di copiare le chiavi SSH sul dispositivo. Le chiavi devono essere inserite sulla penna USB nella cartella "keys", se non è presente la cartella è necessario crearla nella directory principale della penna USB.

  • Menù Time Set. Imposta l’orologio interno indipendente, a batteria, e configura, se collegato ad internet, la sincronizzazione ad un orologio di rete.

  • Menù Network. Permette di cambiare le impostazioni del wifi, ethernet e del mobile.
