Cloner

Il "Cloner" è una utility di recovery per i pannel Mect.

Procedura per l'avvio della recovery:

  • Copiare i file img_cloner_3.x.x.ext2, sysupdate_cloner_3.x.x.sh e _ysupdate_img_3.3.6.sh nella cartella principale di una pen drive USB (formatta FAT32 a singola partizione - ms-dos MBR).
    Attenzione: NON devono essere presenti altri file di aggiornamento "sysupdate_"

  • Spegnere il dispositivo.

  • Inserire la chiavetta USB (se il dispositivo ha una micro-USB, utilizzare un adattatore micro-USB (tipo A) --> USB).

  • Accendere il dispositivo.

Sulla pen drive verrà creata la cartella "cloner" dove saranno inserite le immagini di BACKUP.


  • Effettua il backup dell'applicazione e delle impostazioni correnti. Il backup verrà salvato sulla penna USB nella cartella cloner.

  • Effettua il restore dell'applicazione e delle impostazioni correnti. E' possibile scegliere se effettuare o meno il restore delle retentive ("Ignore / Restore") oppure azzerarle ("Zero"). Il file di restore deve essere sulla penna USB nella cartella cloner.

  • Effettua l'update della Mect Suite. L'immagine di update deve essere inserita nella cartella principale della penna USB e deve essere coerente con il dipositivo da aggiornare. L'icona sarà grigia se non sono presenti Update per il pannello scelto.

  • Factory Reset del pannello. Installa l'applicazione "Simple" del pannello e resetta le impostazioni.

  • Menù VPN. (Disabilitato)

  • Menù SSH. Permette di copiare le chiavi SSH sul dispositivo. Le chiavi devono essere inserite sulla penna USB nella cartella "keys", se non è presente la cartella è necessario crearla nella directory principale della penna USB.

  • Menù Time Set. (Disabilitato)

  • Menù Network. Permette di cambiare le impostazioni del wifi, ethernet e del mobile.
