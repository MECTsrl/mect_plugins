Encoder configuration

TPAC1008 / TPLC100: before configuring, wait until the variable “CH0_01_NODERUN” is at “TRUE”.

TPAC1007 / TPLC050: the configuration can be performed within the “Task0_Init” of the plc software.


Variable

Configuration

PLC_Conf_enc_..

  • 0 encoder disabled

  • 1 counter

  • 2 bidirectional encoder enabled

Ex°: Encoder 3 configured as bidirectional

PLC_Conf_enc_3 := 2;
