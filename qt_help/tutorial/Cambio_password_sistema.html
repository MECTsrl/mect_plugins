Cambio password di sistema

Non usabile dalla versione 3.3.x

Di default lo “Username” e la “Password” di sistema sono:

  • Username: root

  • Password: root

Per cambiare la password seguire la sequenza di istruzioni da scrivere in ssh / telnet:

  • mount -o remount,rw /

  • passwd

  • mount -o remount,ro /

Nel dettaglio seguire i seguenti passaggi:

  • Aprire il “Prompt dei comandi”

  • Scrivere: telnet 192.168.5.211 (indirizzo IP del dispositivo)

  • In “mect login:” scrivere: root

  • In “Password” scrivere: root

  • Scrivere: mount -o remount,rw /

  • Premere “Enter”

  • Scrivere: passwd

  • Premere “Enter”

  • Scrivere la propria password e premere “Enter”

  • Dare conferma della nuova password e premere “Enter”

  • Scrivere: mount -o remount,ro /

  • Premere “Enter”

A questo punto la password di sistema è stata modificata.
