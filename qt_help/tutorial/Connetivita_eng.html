sMily remote connection

It is possible to connect the device through the net LAN, Wi-Fi and Mobile. LAN connectivity is available directly on Ethernet connector and the Mobile and Wi-Fi connection is available using an USB dongle.

sMily remote connection service is available on request.

With the sMily remote connection service it is remotely possible to:

  • Update the HMI program;

  • Update the PLC program;

  • Debug the PLC program (breakpoints);

  • Connect to the device webserver;

  • See and drive the application by VNC.

All that as the device is on its own desktop (if the connection speed allows it).

Moreover it is possible to monitor the application variables.

WARNING: only Wi-Fi and Mobile (3G) key models tested by Mect are guaranteed to be working.
