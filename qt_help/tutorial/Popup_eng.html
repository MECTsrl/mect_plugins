Add a popup window

It is possible to make appear a popup window when pressing an ATCMbutton. The window can be, for example, an “information” or “warning” icon or a “critical” message.

Please follow “User function” and add the following code:

QMessageBox::information(this,trUtf8(“USB error”), trUtf8(“Cannot mount the usb key”));

The following example is an “information” icon

USB error is the title and Cannot mount the usb key è is the message.

Insert the following code at the beginning of pagexxx.cpp:

	  #include <QMessageBox>

To show a “warning” icon please write:

QMessageBox::warning(this,trUtf8(“USB	error”), trUtf8(“Cannot mount the usb key”));

To show a “critical” message please write:

QMessageBox::critical(this,trUtf8(“USB	error”), trUtf8(“Cannot mount the usb key”));
