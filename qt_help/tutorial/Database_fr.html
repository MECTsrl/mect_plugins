Accès aux bases de données MySQL/SQLite

Pour accéder aux bases de données MySQL/SQLite, ajoutez ce qui suit au fichier de projet(.pro)

   QT += sql
   QTPLUGIN += qsqlite

Saisissez le code suivant, par exemple dans automation.cpp (les contrôles d'erreur manquent):

	  #include <QtSql> 
	  #include <QSqlDatabase> 
	  #include <QDebug>
	  #include <QSqlQuery>	
	  QSqlDatabase db; //Database object

Dans le setup()

	  db = QSqlDatabase::addDatabase("QSQLITE", "ConnectionName");
    db.setDatabaseName("/local/flash/root/test.sqlite");
    if (!db.open()) {
        qDebug() << QString("Error occurred opening the database. %1") .arg(db.databaseName());
        qDebug() << QString("%1.")  .arg(db.lastError().text());
    }
    else  {
        qDebug() << QString("Data base %1 Opened") .arg(db.databaseName());
    }

Le premier paramètre de addDatabase indique quel driver utiliser (dans ce cas le driver SQLite), le second donne un nom à la connexion, et est utilisé pour ouvrir plus d'une base de données en même temps.

Dans le loop()

{
   QString testo = "SELECT * FROM TABELLA_X" ;
   QSqlQuery query(testo, db); 
   testo += "\n";
    if (!query.exec()) { 
       testo += QString("Open error [%1]").arg(db.lastError().text());
    } else { 
       for (int n = 0; n < query.record().count(); n++)
       { 
           testo += query.record().fieldName(n) + " - "; 
       } 
       testo += "\n";
	
       while(query.next()) { 
               for (int n = 0; n < query.record().count(); n++) 
               { 
                   testo += query.value(n).toString() + " | ";
	
               } 
               testo += "\n";
       } 
   }
   query.finish(); 
   // 
}

Dans le code vous devriez fermer le fichier, ou plutôt le code d'ouverture de la BDD ne devrait être exécuté que si nécessaire et le fichier devrait être fermé immédiatement après, dans une séquence comme celle-ci.

// Closing DB 
    if (db.isOpen())
        db.close();

	ln –s /usr/lib/libsq* ./
