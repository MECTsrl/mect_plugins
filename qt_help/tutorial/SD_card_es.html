Inserción de la tarjeta micro SD

En los paneles TPAC1007_04 es posible insertar una tarjeta micro SD de hasta 64 GB (formato ext3 o FAT con una sola partición). La primera vez que entre, aparecerá la siguiente pantalla:

Puede elegir entre 4 modos:

  • Remind me at next boot: esta pantalla se mostrará de nuevo la próxima vez que inicies el panel de operador.

  • Ignore it: ignora la inserción de la tarjeta y ya no mostrará esta pantalla en el siguiente reinicio dejando la tarjeta montada.

  • Use it as main storage (HMI+PLC+LOG): la tarjeta se utiliza como memoria para guardar la aplicación HMI, PLC incluyendo cualquier LOG. Solo con tarjeta SD formateada en ext3.

    ATENCIÓN: Si retira la tarjeta, la aplicación ya no estará presente en el panel de operador. Si mueve la tarjeta a otro panel del mismo modelo, se iniciará con la aplicación contenida en la tarjeta SD.

  • Use it as data storages (only LOG): La tarjeta se utiliza exclusivamente como memoria para LOGs. Tamaño máximo 4GB.


En los dos últimos casos, solo si la tarjeta SD ha sido formateada en formato ext3, también es posible crear un archivo de intercambio para aumentar la memoria RAM disponible.

Debe marcar la casilla de verificación "crear archivo de intercambio de memoria" y el tamaño del área deseada (16/32/64/128/256 MB).

La elección propuesta es de 128 MB, igual a la RAM presente en la tarjeta.

Tras seleccionar el tipo de uso deseado, si se marca esta opción, se iniciará la preparación de la zona de Intercambio.

Esto puede llevar mucho tiempo (alrededor de 12 minutos para 128 MB) y solo se realizará una vez.

El uso de la memoria RAM y el área de intercambio se puede verificar con el comando "liberar" desde la terminal.
