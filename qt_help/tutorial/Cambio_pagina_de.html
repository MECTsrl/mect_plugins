Seite wechseln

Um die Schaltfläche zum Ändern einer Seite zu verwenden, führen Sie diese Schritte aus:

  1. Ziehen Sie das Objekt auf die Seite.

  2. Wählen Sie die Schaltfläche und geben Sie in die Eigenschaft page_Name (rechte Seite) den Namen der Seite ein, zu der Sie gehen möchten(falls Sie zur Startseite gehen wollen, schreiben Sie einfach HOME, falls Sie zur zuletzt besuchten Seite zurückkehren wollen, schreiben Sie einfach BACK). Wenn Sie Systemseiten verwenden möchten, sehsen Sie den Abschnitt “Bibliotheks-Seiten”.

  3. Kreuzen Sie an remember-Eigenschaft , wenn Sie die Zielseite zum Seitenverlauf hinzufügen möchten, damit Sie die BACK-Funktion später nutzen können.
