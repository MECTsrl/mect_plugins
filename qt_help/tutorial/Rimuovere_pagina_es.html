Eliminar una página

Para eliminar una página existente, entre en la sección “Form” del árbol de proyecto a la izquierda y haga clic con el botón derecho del ratón en el archivo.ui relacionado con la página que desea eliminar y seleccione la opción “Remove file”. En la siguiente ventana seleccione “Delete file permanently” e pulse “Close”.

Atención: después de eliminar la página deseada debe “limpiar” el proyecto para evitar errores durante la compilación.

En particular, pulse Build → Clean All

Espere a que la barra se ponga verde.

Luego pulse Build → Run qmake

Espere a que la barra se ponga verde.

Atención: Haga otra vez Build en caso de que se reporte un error.
