Cambiar la contraseña del sistema

Por defecto el “Username” y la “Password” del sistema son:

  • Username: root

  • Password: root

Para cambiar la contraseña siga la secuencia de instrucciones que se escribirán en ssh / telnet:

  • mount -o remount,rw /

  • passwd

  • mount -o remount,ro /

Siga los pasos que se detallan a continuación:

  • Abra el “símbolo del sistema”

  • Escriba: telnet 192.168.5.211 (dirección IP del dispositivo)

  • En “mect login:” Escriba: root

  • En “Password” Escriba: root

  • Escriba: mount -o remount,rw /

  • Pulse “Enter”

  • Escriba: passwd

  • Pulse “Enter”

  • Introduzca su contraseña y pulse “Enter”

  • Confirme la nueva contraseña y pulse “Enter”

  • Escriba: mount -o remount,ro /

  • Pulse “Enter”

Se ha modificado la contraseña del sistema.

Para transferir la aplicación HMI es necesario cambiar la contraseña asociada al programa en QtCreator. Siga la imagen de abajo.
