Zugriff auf MySQL/SQLite-Datenbanken

Um auf MySQL/SQLite-Datenbanken zuzugreifen, fügen Sie Folgendes zur Projektdatei(.pro) hinzu

   QT += sql
   QTPLUGIN += qsqlite

Geben Sie folgenden Code ein, z.B. in der Datei automation.cpp (Fehlerprüfungen fehlen):

	  #include <QtSql> 
	  #include <QSqlDatabase> 
	  #include <QDebug>
	  #include <QSqlQuery>	
	  QSqlDatabase db; //Database object

Innerhalb des setup()

	  db = QSqlDatabase::addDatabase("QSQLITE", "ConnectionName");
    db.setDatabaseName("/local/flash/root/test.sqlite");
    if (!db.open()) {
        qDebug() << QString("Error occurred opening the database. %1") .arg(db.databaseName());
        qDebug() << QString("%1.")  .arg(db.lastError().text());
    }
    else  {
        qDebug() << QString("Data base %1 Opened") .arg(db.databaseName());
    }

Der erste Parameter von addDatabase sagt, welcher Treiber verwendet werden soll (in diesem Fall der SQLite-Treiber), der zweite gibt der Verbindung einen Namen, und er wird verwendet, um mehr als eine DB gleichzeitig zu öffnen.

Innerhalb des loop()

{
   QString testo = "SELECT * FROM TABELLA_X" ;
   QSqlQuery query(testo, db); 
   testo += "\n";
    if (!query.exec()) { 
       testo += QString("Open error [%1]").arg(db.lastError().text());
    } else { 
       for (int n = 0; n < query.record().count(); n++)
       { 
           testo += query.record().fieldName(n) + " - "; 
       } 
       testo += "\n";
	
       while(query.next()) { 
               for (int n = 0; n < query.record().count(); n++) 
               { 
                   testo += query.value(n).toString() + " | ";
	
               } 
               testo += "\n";
       } 
   }
   query.finish(); 
   // 
}

Im Code sollten Sie die Datei schließen, bzw. der Öffnungscode der DB sollte nur bei Bedarf ausgeführt werden und die Datei sollte unmittelbar danach in einer solchen Reihenfolge geschlossen werden.

// Closing DB 
    if (db.isOpen())
        db.close();

Zusätzlich (für kleinere Nebenversionen von MectSuite 3.3.x) müssen Sie einen Ordner auf dem TPAC “sqldrivers” hinzufügen, der die symbolischen Links zu den DB-Interface-Plugins enthält:

	mkdir sqldrivers
	cd sqldrivers
	ln –s /usr/lib/libsq* ./
	ln –s /usr/lib/libqsqlite* ./
	cd
