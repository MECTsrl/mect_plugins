Fonctionnalité de la Crosstable: Recherche

En appuyant sur le bouton de recherche, la fenêtre de recherche avancée apparaîtra.

Dans la fenêtre, il est possible d'affiner la recherche en sélectionnant les paramétres qui composent les variables dans la Crosstable.


Le bouton Reset réinitialise tous les paramàtres de recherche.

Le bouton OK permet d'accéder á la variable sélectionnée dans la Crosstable.

Le bouton Cancel ferme la fenêtre.
