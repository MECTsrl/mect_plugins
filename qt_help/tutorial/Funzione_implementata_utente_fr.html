Fonction implémentée par l'utilisateur

Si vous souhaitez associer la pression du bouton à l'exécution d'une fonction créée par l'utilisateur, procédez comme suit:

  1. Faites glisser l'objet sur la page.

  2. Cliquez avec le bouton droit (de la souris) sur le bouton et sélectionnez “go to slot”.

    Si la fenêtre suivante apparaît:

    Vous devez faire un “Build All” puis fermer le projet. Lorsque le projet est rouverte, répétez l'opération à partir du point 2.

  1. Implémenter la fonction en langage C++ dans la fenêtre qui s'est ouverte à l'étape précédente.
