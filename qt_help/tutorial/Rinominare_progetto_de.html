Ein Projekt umbenennen

Um den mit QtCreator erstellten Projektordner umzubenennen, müssen Sie die folgenden Schritte ausführen:

  1. Stellen Sie sicher, dass Sie das Projekt, das Sie umbenennen möchten, geschlossen haben.

  2. Benennen Sie den Projektordner um (fügen Sie keine Sonderzeichen (außer _) oder Leerzeichen ein).

  3. Löschen Sie die Datei mit der Erweiterung “.pro.user” aus dem Projektordner.

  4. Benennen Sie die Datei mit der Erweiterung “.pro” mit dem Namen, den Sie in Schritt 2 ausgewählt haben, um.

  5. Benennen Sie die Datei mit der Erweiterung “.pro.shared” mit dem Namen, den Sie in Schritt 2 ausgewählt haben, um.

  6. Öffnen Sie das neue Projekt von QtCreator.

  7. Wenn das folgende Bild erscheint, drücken Sie die Taste “Configure Project”.

  1. Führen Sie einen “Run qmake” und einen “Clean All” aus, siehen Sie Kapitel “Kompilierungsfehler”

  2. Machen Sie das “Build” des Projekts.

  3. Legen Sie die Parameter für das Senden des Projekts an das Gerät fest:

    • Deaktivieren Sie , unter der Überschrift “Upload files via SFTP”, das flag “Incremental deployment”;

    • Klicken Sie auf “Add Deploy Step” und Wählen Sie “Run custom remote command”

    • Schreiben Sie “stop” (alles Kleinbuchstaben) und verschieben Sie den “Run custom remote command” Block um eine Position nach oben, Verwenden Sie den gelb markierten Pfeil (drücken Sie Details).
