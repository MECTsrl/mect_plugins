Modifier le mot de passe du système

Non utilisable depuis la version 3.3.x

Par défaut le “Username” et la “Password” du système sont:

  • Username: root

  • Password: root

Pour changer le mot de passe, suivre la séquence des instructions à écrire en ssh / telnet:

  • mount -o remount,rw /

  • passwd

  • mount -o remount,ro /

Suivez les étapes ci-dessous en détail:

  • Ouvrez l'“Invite de commandes”

  • Ecrivez: telnet 192.168.5.5.211 (adresse IP du périphérique)

  • Dans “mect login:” écrivez: root

  • Dans “Password” écrivez: root

  • Ecrivez: mount -o remount,rw /

  • Appuyez sur “Enter”

  • Ecrivez: passwd

  • Appuyez sur “Enter”

  • Entrez votre mot de passe et appuyez sur “Enter”

  • Confirmez le nouveau mot de passe et appuyez sur “Enter”

  • Ecrivez: mount -o remount,ro /

  • Appuyez sur “Enter”

Le mot de passe du système a été modifié.

Pour transférer l'application IHM, vous devez changer le mot de passe associé au programme dans QtCreator. Suivez l'image ci-dessous.
