Trend anzeigen

Fügen Sie einen ATCMbutton ein und geben Sie in die Eigenschaft page_Name (rechte Seite) den Namen der Seite des trend, Standard trend1.

Die Grafik funktioniert insbesondere in zwei Modi:

  • online: Die Grafik scrollt automatisch, so dass die letzten Proben immer sichtbar sind;

  • offline: Die Grafik ist statisch und selbst wenn neue Proben zur Anzeige verfügbar sind, werden diese nur angezeigt, wenn sie im aktuellen Anzeigebereich enthalten sind.

Zu Beginn sehen Sie die Grafiken, aus denen sich die Familie zusammensetzt, in Echtzeit (online).

Die Trendfamilien bestehen aus bis zu 4 Grafiken. Hier ist die Beschreibung der Schaltflächen auf der rechten Seite:

Unten links zeigt ein Label den Wert der Variablen an, auf die sich der Trend bezieht.

Auf der linken Seite der Grafik befindet sich eine Schaltfläche mit der Sie die Grafik auswählen können, die Sie im Vordergrund haben möchten (die Farbe ändert sich je nach gewählter Grafik). Automatisch wird die Achse der Werte der aktuellen Variable angezeigt und auf der Schaltfläche unten wird die Beschreibung der aktuellen Grafik angezeigt (Z.b Analog Input 1).

Um die Einstellungen die Grafiken zu ändern, können Sie den “Trends Editor” ( später machen Sie das deploy vom Projekt aus QtCreator ) verwenden oder direkt vom Gerät aus durch Drücken der Taste unten / in der Mitte (im Falle des ersten Bildes des Kapitels die Worte “Analog Input 1”).

Für jede Grafik müssen Sie angeben, ob sie sichtbar ist oder nicht, die Variable, auf die sie sich bezieht, eine mögliche Beschreibung, die Farbe, die Sie zuordnen möchten, und den Wertebereich, der von der Variable angenommen wird (Ymin/Ymax).

Grafiken können sowohl mit der horizontalen (landscape) als auch mit der vertikalen (portrait) Zeitachse betrachtet werden.

Die Schaltflächen können den Trend und/oder die Variable, die innerhalb des Trends angezeigt werden soll, eliminieren.

Sobald sich die Werte der Grafiken geändert haben, drücken Sie einfach die Schaltfläche Save wenn Sie die aktuelle Familie überschreiben möchten, Save as wenn Sie eine neue Trendfamilie erstellen möchten, die aus der Grafik mit den vorgenommenen Änderungen und den anderen Grafiken der Quellfamilie besteht, Save to USB wenn Sie die Trendfamilie auf dem USB-Stick speichern möchten. Die auf dem USB-Stick gespeicherte Datei ähnelt der im Trends Editor gespeicherten Datei.

Durch Drücken der Schaltfläche können Sie einen neuen Trend erstellen, als ob Sie den Trend Editor verwenden würden.

ATTENZIONE: Alle Änderungen, die direkt am Gerät vorgenommen werden, sind im Trend-Editor nicht sichtbar , es sei denn, Sie speichern auf USB und importieren den neuen Trend aus dem Trend-Editor (mit der Taste Import Trend).
