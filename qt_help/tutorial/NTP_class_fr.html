La classe NTP Client

La classe ntpclient est une librairie de synchronisation avec le protocole NTP qui peut être incluse dans vos projets.

Les fonctions qui peuvent être utilisées sont :
  1. QString getNtpServer();
  2. int getTimeout_s();
  3. int getOffset_h();
  4. int getPeriod_h();
  5. void setNtpParams(const QString &ntpServer, int ntpTimeout_s, int ntpOffset_h, int ntpPeriod_h);
  6. void requestNTPSync();
  7. void requestDateTimeChange(QDateTime newTime);

Dans l'exemple de projet time_set_AnyTPAC043 dans mect_apps/Exemples, toutes ces fonctions sont utilisées pour recréer la page système Time Set.
