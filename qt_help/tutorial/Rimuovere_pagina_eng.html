Remove an existing page

To remove an existing page, access the “Form” section in the project tree on the left and click the right mouse key on .ui file corresponding to the page to remove and choose item “Remove file”. Select “Delete file permanently” and press “Close” in the next window.

Remove the relevant .cpp and .h page.

WARNING: after removing the desired page must make a “cleaning” of the p project to avoid errors during compilation.

Press Build → Clean All

Wait until the bar turns green.

Then press Build → Run qmake

Wait until the bar turns green.

WARNING: Build again if there is a error.
