Personnaliser l'arrière-plan d'une page

Vous pouvez personnaliser l'arrière-plan d'une page en changeant sa couleur, en ajoutant une image et en plaçant une bordure.

Pour ce faire, procédez comme suit:

  1. Cliquez avec le bouton droit de la souris sur un point libre de la page

  2. Choisissez l'option “Change styleSheet”

  3. L'image suivante apparaîtra:

  1. Placez le curseur entre les crochets bouclés et sélectionnez dans le cas de:

    • une image: “Add Resource → image”

    • une couleur: “Add Color → background-color”

NOTE: Il est fortement déconseillé d'utiliser le dégradé pour définir l'arrière-plan d'une page car l'appareil serait lent. Utilisez une seule couleur ou une seule image pour surmonter cette situation.


Personaliser un splash screen

Pour personnaliser un spalsh screen, il suffit d'écraser le fichier “splash.png” dans le dossier “config” du projet. L'image du splash doit être au format png, de la même résolution que l'écran du produit utilisé:

  • TPAC1006/TP1057 => 320x240;

  • TPAC1007/TP1043 => 480x272;

  • TPAC1008/TP1070 => 800x480.
