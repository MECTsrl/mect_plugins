Changelog

MectSuite release notes.

[v3.3.4] - 2021-03-22

  • Security Features

    System:
    - Hardening is enabled by default for all machines (#339).
    - Sysupdate now enables Hardening by default (#8).

  • New Features

    Models:
    - New TPAC1007_04_AD and TPAC1007_04_AE (#52).
    - New TPX1070_03_D and TPX1070_03_E (#215).
    Crosstable Editor:
    - Advance variables search (#100).
    - New multi line edit mode (#326).
    - New 'AnyTPAC043' and 'AnyTPAC070' project templates (#60).
    System:
    - New NTP feature in Time Set page and ntpclient class (#228).
    - Added special ASCII symbols in alphapad (#68).
    - New empty line '-;-;-' in logs when date and time is manually changed and when NTP sync happens (#97).
    MectApps:
    - New 4C_TestSuite (#90).
    Utilities:
    - New Sysupdates for IP and touch screen calibration reset (#19).
    - New get_trends.cgi (#31).
    - First µPython sysupdate release. (#)

  • Fixed Issues

    Crosstable Editor:
    - The Combobox section now follows the current line during the scroll (#110).
    - "Undo" button now works well (#242).
    - Various fixes in Crosstable and Ui Designer (#109).
    - Fix wrong behavior while editing a model not included in the model list (#243).
    - Fix TPAC100803XX Tooltip (#71).
    - RTU now accept Node ID from 1 to 247 (#39).
    - TCP_SRV and RTU_SRV now checks the next register if a variable is 32bit (#40).
    - Added range control (0 to 4095) for Modbus Server variables offset (#205).
    - Fixed variable decimals control error if specified by another variable (#112).
    - Fixed Error Code 24, can not create a new alarm variable (#137).
    - Fix MPNC ColdJunctionT variable (#83).
    System:
    - Avoided CMD_GETCHILDREN crash (#82).
    - Fixed Timers (ex: TON) overflow (#87).
    - ATCM animation now accepts PNG with uppercase extension (#47).
    - Splash screen now is correctly closed if the RTC battery is down and date and time is set in Time Set page (#108).
    - Workaround for next line while ATCMButton is Pressed (#17).
    - JSON from get_log.cgi now displays the values in the correct order (#81).
    - Minor Bug fix (#85, #89, #104, #88)
    Cloner:
    - Cloner now adds underscore instead of white space while create new directory (#94).

  • Changed Features

    Crosstable Editor:
    - Max logs size can grow up to 4096 MB (for SDCard) (#65).
    - Input Registers can be defined in Modbus TCP, TCP_SRV and RTU_SRV (#113).
    - RTU_SRV node number can be modified (#101).
    - RTU0 fields are no longer editable by users in TPAC100803XX (#73).
    - Changed behavior of MPNE_AnIn1Conf and MPNE_Watchdog for MPNE (#69).
    - Changed behavior of PLC_EncoderCo_Lo/Hi with no decimal for TPAC100802XX e TPAC100803YY (#70).
    - Changed some variables name (PLC_Pwm_* to PLC_Pto_*) in TPAC100803XX Crosstable Editor (#72).
    - Changed some variables name (MPNE_PTORampUp_* to MPNE_ControlWord_*) in MPNE1001 Crosstable Editor (#78).
    - Changed paste menu text from "Paste MPNE10 Module" to "Paste MPNE1001 Module" (#79).
    - Changed priority from 2 to 1 and the port from 0 to 3 when Paste TPLC050 Modules (#58).
    - Changed PLC_EncoderCounter from UDINT to DINT in TPLC050 (#61).
    - TPAC1007_04_AE default RTU3 baudrate is set to 38400 and address 20 (14hex) is not permitted (#231).
    - Multiple copy of one line over multiple non contigues line is now permitted (#111).
    System:
    - HMI Network page is now able to connect to hidden wifi networks (#66).
    - HMI Time Set page Review (#307).
    - HMI Store Page review (#387).
    - CGI Review (#222).
    - Minor internal changes (#56).

  • Removed

    Models:
    - TP1070_02_E (#252).
    - TP1057 (#34).
    - TP1043_01_B (#34).
    - TPAC1006 (#34).
    - TPAC1008_01 (#34).
    Crosstable Editor:
    - Disabled RTU3 for TPAC1007_LV and TPAC1007_04_AE (#80).
    System:
    - Removed old emails from source code (#102).

  • Know Issues

    System:
    - Numpad on UDINT values can write a number up to 429496729.
