MectSuite v3.3.7 release notes.

What's New

v3.3.7 [2021-05-06]

  • New Features

    Models

    - New TPX1043_03_C.

  • Fixed Issues

    Crosstable Editor

    - Fix Crosstable Editor crash if no trend found in Project.
    - Minor Fixes.

    System

    - Fix H variables reading error.
    - Fix Modbus node id check.
    - Minor Fixes.

    Cloner

    - Minor Fixes.

  • Changed Features

    Crosstable Editor

    - Added Red/Green foreground for Timings column in Devices and Timings tree.
    - Added Logged Variables counter in "Logged Vars" Tab.
    - Added "Simplify Diagnostic Vars" button.

    System

    - HMI-PLC protocol optimization.
    - New alphapad with showSpecialChars option.
    - Numpad now can write unsigned integers.

    Cloner

    - Added more restore options.


v3.3.4 [2021-03-23]

  • Security Features

    System

    - Hardening is enabled by default for all machines. For more info check the Help page "Download to the panel of a graphic application"
    - Sysupdate now enables Hardening by default.

  • New Features

    Models

    - New TPAC1007_04_AD and TPAC1007_04_AE.
    - New TPX1070_03_D and TPX1070_03_E.

    Crosstable Editor

    - Advanced variables search.
    - New multi line edit mode.
    - New 'AnyTPAC043' and 'AnyTPAC070' project templates.

    System

    - New NTP feature in Time Set page and ntpclient class.
    - Added special ASCII symbols in alphapad.
    - New empty line '-;-;-' in logs when date and time is manually changed or when NTP sync happens.

    MectApps

    - New 4C_TestSuite.

    Utilities

    - New Sysupdates for IP and touch screen calibration reset.
    - New get_trends.cgi.
    - First µPython sysupdate release.

  • Fixed Issues

    Crosstable Editor

    - The Combobox section now follows the current line during the scroll.
    - "Undo" button now works well.
    - Various fixes in Crosstable and Ui Designer.
    - Fix wrong behavior while editing a model not included in the model list.
    - Fix TPAC100803XX Tooltip.
    - RTU now accept Node ID from 1 to 247.
    - TCP_SRV and RTU_SRV now checks the next register if a variable is 32bit.
    - Added range control (0 to 4095) for Modbus Server variables offset.
    - Fixed variable decimals control error if specified by another variable.
    - Fixed Error Code 24, can not create a new alarm variable.
    - Fix MPNC ColdJunctionT variable.
    - Minor fixes.

    System

    - Avoided CMD_GETCHILDREN crash.
    - Fixed Timers (ex: TON) overflow.
    - In Store Page, HMI is no longer freezed while saving logs in USB .
    - ATCM animation now accepts PNG with uppercase extension.
    - Splash screen now is correctly closed if the RTC battery is down and date and time is set in Time Set page.
    - Workaround for next line while ATCMButton is Pressed.
    - JSON from get_log.cgi now displays the values in the correct order.
    - Minor fixes.

    Cloner

    - Cloner now adds underscore instead of white space while create new directory.
    - Cloner now ask for confirmation before restoring an image.

  • Changed Features

    Crosstable Editor

    - Max logs size is set by default to 4096 MB.
    - Input Registers can be defined in Modbus TCP, TCP_SRV and RTU_SRV.
    - RTU_SRV node number can be modified.
    - RTU0 fields are no longer editable by users in TPAC100803XX.
    - Changed behavior of MPNE_AnIn1Conf and MPNE_Watchdog for MPNE.
    - Changed behavior of PLC_EncoderCo_Lo/Hi with no decimal for TPAC100802XX e TPAC100803YY.
    - Changed some variables name (PLC_Pwm_* to PLC_Pto_*) in TPAC100803XX.
    - Changed some variables name (MPNE_PTORampUp_* to MPNE_ControlWord_*) in MPNE1001.
    - Changed paste menu text from "Paste MPNE10 Module" to "Paste MPNE1001 Module".
    - Changed priority from 2 to 1 and the port from 0 to 3 when Paste TPLC050 Modules.
    - Changed PLC_EncoderCounter from UDINT to DINT in TPLC050.
    - TPAC1007_04_AE default RTU3 baudrate is set to 38400 and address 20 (14hex) is not permitted .
    - Multiple copy of one line over multiple non contigues line is now permitted.

    System

    - HMI Network page is now able to connect to hidden wifi networks.
    - HMI Time Set page Review.
    - CGI Review.
    - Internal changes.

  • Deprecated

    Models

    - TP1070_02_E.
    - TP1057.
    - TP1043_01_B.
    - TPAC1006.
    - TPAC1008_01.

  • Removed Features

    Crosstable Editor

    - Disabled RTU3 for TPAC1007_LV and TPAC1007_04_AE.

    System

    - Removed old emails from source code.
