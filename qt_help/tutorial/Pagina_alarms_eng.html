Alarms page

To show alarms page use an ATCMbutton and insert in the property “pageName” word “alarms”. See chapter “Library pages”.

On top there one combobox where to select the type (All, Alarm and Event). These selections are used for alarms shown in the lower list. In this object alarms actually visible are shown.

The alarms can be displayed in 3 different colors:

  • Red: alarm condition still present.

  • Blue: the operator has acknowledged (ACK) but the alarm condition is still present.

  • Gray: alarm condition returned. The alarm display will disappear after acknowledging (ACK).

Below there is a banner that shows in turn the actually visible alarms.


On bottom, near keys Home and Back there are also the following keys:

  1. key to acknowledge a selected alarm in the above list;

  2. key to acknowledge all visible alarm;

  3. key to disable alarm buzzer;

  4. key to show history alarm page.
