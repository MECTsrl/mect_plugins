Alarms page

On top there one combobox where to select the type (All, Alarm and Event). These selections are used for alarms shown in the lower list. In this object alarms actually visible are shown. Below there is a banner that shows in turn the actually visible alarms.

On bottom, near keys Home and Back there are also the following keys:

  1. key to acknowledge a selected alarm in the above list;

  2. key to acknowledge all visible alarm;

  3. key to disable alarm buzzer;

  4. key to show history alarm page.
