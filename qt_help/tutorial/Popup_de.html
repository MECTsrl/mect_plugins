Einfügen eines Popup-Fensters

Es ist möglich, dem Drücken einer ATCM-Taste die Visualisierung eines Popup-Fensters zuassoziieren. Dieses Fenster kann beispielsweise ein Symbol für “Information”, “Warnung” oder “critische” Meldung enthalten.

Folgen Sie dem Kapitel “ Vom Benutzer implementierte Funktion” und geben Sie den folgenden Code ein:

QMessageBox::information(this,trUtf8(“USB error”), trUtf8(“Cannot mount the usb key”));

In diesem Fall wird beim Drücken der Taste ein Fenster mit dem Symbol “Information” angezeigt.

Im Beispiel entspricht USB error dem Titel, während das Schreiben Cannot mount the usb key die Nachricht ist.

Um das Symbol “Warnung” anzuzeigen, schreiben Sie:

QMessageBox::warning(this,trUtf8(“USB	error”), trUtf8(“Cannot mount the usb key”));

Um das Symbol für die “kritische” Nachricht anzuzeigen, schreiben Sie:

QMessageBox::critical(this,trUtf8(“USB	error”), trUtf8(“Cannot mount the usb key”));
