Aggiungere immagini / font

Affinché un elemento esterno al progetto (immagini etc...) possa essere visto e incluso nel progetto, bisogna aggiungerlo al file di risorse. Per fare questo seguire i seguenti passi:

  • Inserire la propria immagine / font, attraverso esplora risorse, all’interno della cartella “icons” o “fonts” nella cartella di progetto;

  • Fare doppio click al file .qrc nella sezione “Resources” che si trova nel riquadro della struttura del progetto;

  • Premere il pulsante “Add” per aggiungere una nuova risorsa;

  • Salvare il file resources.qrc.

A questo punto l’oggetto inserito può essere visto all’interno del progetto.
