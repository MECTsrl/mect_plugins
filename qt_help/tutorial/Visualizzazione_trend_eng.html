Trend display

Trend works in 2 ways:

  • online: graph shifts automatically in order to see always last samples;

  • offline: graph is static even new samples are available, they will be shown only if in the actual display window.

At beginning traces of the family are shown in real time (on-line).

Trend families are composed of 4 traces maximum. Here the description of right size keys:

Home


Back


Time edit: shows a specific area numerically defined


Pan: resets zooms and axis change scales


Activates zoom


Shows another keyboard

  • Time edit, key opens a window where modify time axis range (for all traces) and value axis range (of the active trace). These values will not be saved, so are lost when actual trend is closed.

  • To zoom an area, enter the zoom mode and select an area with touchscreen.

  • key shows another keyboard (pressing key again the keyboard disappears):

shift forward time axis


shift back time axis


shift up values axis


shift down values axis


On/off line: enable/disable online mode

saves on panel the actual screenshot

    • Left and right commands disable online mode.

    • Up and down key are enabled only if there are values outside the displayed range.

In the left down corner a label shows the trend value.

On the left of graphic there is a button to select the trace to have in foreground (color changes according to selected trace). Automatically the value axis of the actual variable is shown and on the bottom button a description of the trace appears (ex° Analog Input 1).

To change traces setup it is possible to use the “Trends Editor” (and then do again the deploy of the QtCreator project) or directly on the device using the central bottom side (in the first image of the chapter, the “Analog Input 1”).

For each trace specify if it is visible or not, the relevant variable, a description, its color and the range for values of the variable (Ymin/Ymax).

Traces are visible in landscape or in portrait .

Keys can cancel the trend and/or the variable inside the trend.

After changing the values of the trace press Save key if you want to update the actual family, or Save as if you want to create a new trend family, that is formed by the modify trace and the other traces of the original family, Save to USB save the trend family on the USB pen. File on USB pen is similar to file saved with Trends Editor.

Key creates a new trend, in the same way of using Trends Editor.

WARNING: all changes done on the device are not visible in the Trends Editor unless perform a Save to USB and import the new trend in the Trends Editor (with key Import Trend).
