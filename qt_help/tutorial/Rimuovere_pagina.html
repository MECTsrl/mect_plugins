Rimuovere una pagina

Per rimuovere una pagina esistente, entrare nella sezione “Form” dall’albero del progetto sulla sinistra e premere il pulsante destro del mouse sul file .ui relativo alla pagina da eliminare e scegliere la voce “Remove file”. Nella finestra successiva selezionare “Delete file permanently” e premere “Close”.

Rimuovere la relativa pagina .cpp e .h

Attenzione: dopo aver rimosso la pagina desiderata occorre fare una “pulizia” del progetto per evitare di avere errori durante la compilazione.

In particolare premere Build → Clean All

Attendere che la barra diventi verde.

Successivamente premere Build → Run qmake

Attendere che la barra diventi verde.

Attenzione: fare un altro Build nel caso in cui venga segnalato un errore.
