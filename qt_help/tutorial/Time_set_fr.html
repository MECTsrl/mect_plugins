Time Set

La page Time Set permet de régler la date et l'heure du système.

Elle est divisée en deux sections :
  1. Manual
  2. NTP
    La fonctionnalité NTP doit être utilisée expressément avec le service sMily activé.

Manual:
Dans la section Manuel, vous pouvez régler manuellement la date et l'heure du système.
En appuyant sur le bouton avec la date, le calendrier apparaîtra.

Pour sélectionner le jour à régler, il suffit d'appuyer sur le jour souhaité.


Il est possible d'avancer ou de reculer dans les mois grâce aux flèches placées en haut du calendrier, ou en appuyant sur le bouton avec le mois en cours, il est possible de visualiser tous les mois de l'année.
Pour changer l'année, vous pouvez avancer ou reculer avec les flèches ou appuyer sur le bouton de l'année en cours et sélectionner l'année souhaitée.

En appuyant sur le bouton avec l'heure, le sélecteur des heures, minutes et secondes apparaît.

En appuyant sur OK, vous sauvegardez les heures, minutes et secondes sélectionnées.
Cancel annule l'opération.

Une boîte de message de confirmation apparaîtra à la fin de l'opération.

NTP:
Dans la section NTP, il est possible de définir via NTP la date et l'heure du système, en particulier il est possible de définir les paramètres pour activer le Network Time Protocol, c'est-à-dire le serveur ntp, le délai en secondes, le décalage en heures et la période en heures.
La fonctionnalité NTP doit être utilisée expressément avec le service sMily activé.

Le serveur NTP par défaut est tempo.ien.it et il est possible de le modifier en cliquant sur le bouton:


Le décalage horaire définit la différence de fuseau horaire et est par défaut de +1 heure (CET). Les valeurs entières de -12 à +12 sont autorisées.


La période définit toutes les heures où la synchronisation NTP doit avoir lieu, 0 h signifie pas de synchronisation, la valeur par défaut est 0


Le délai indique le temps d'attente (en secondes) dans lequel vous pouvez recevoir une réponse du serveur NTP, la valeur par défaut est de 10
