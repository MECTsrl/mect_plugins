Mise à jour

Avant la mise à jour, lisez le fichier "changelog.txt" sur le site Web de Mect dans “DOWNLOAD Mect Suite”.

Pour mettre à jour, téléchargez les fichiers du site mect.it, en particulier: Software and Services => Mect Suite => DOWNLOAD Mect Suite => Stable release 3.x.x => Mect Suite 3.x.x => Update Device

ATTENTION: NE PAS ÉTEINDRE LE SYSTEME PENDANT LA MISE A JOUR.

Mise à jour de l’appareil:

  • img_sysupdate-3.x-NomProduit.ext2 et sysupdate_img_3.x.sh:

    • Via USB: copiez les fichiers “img_sysupdate-3.x-NomProduit.ext2” et “sysupdate_img_3.x.sh” sur une clé USB.

      Par exemple, pour le TPAC1007_04_AC vous devrez copier les fichiers sur la clé:

      • img_sysupdate-3.0.0-TPAC1007_04_AC.ext2

      • sysupdate_img_3.0.0.sh

    Pour la version 3.x cela prend environ 2 minutes. Après l'affichage du message: “Please power off and remove the USB storage device” vous pouvez éteindre l'appareil, retirer la clé USB, puis redémarrer. Pour les modèles TPAC1007 et TP1043, l'adaptateur micro-USB (type A) - USB est requis.

Appareils sans moniteur: attendez environ 4 minutes avant d'éteindre l'appareil.

ATTENTION:Depuis de la version 3.0.0, il n'est plus possible faire la mise à jour via Browser.
