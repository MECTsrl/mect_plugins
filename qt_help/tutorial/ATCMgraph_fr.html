ATCMgraph

Cet objet permet l'affichage d'un graphique à deux pistes.

Pour l'utiliser, suivez les étapes ci-dessous:

  1. Faites glisser l'objet sur la page et double-cliquez.

  2. Définissez la variable x1Variable avec le nom de la variable de la Crosstable à partir de laquelle vous voulez lire la valeur utilisée pour représenter l'abscisse de la piste 1. Appuyez sur le bouton “…” pour sélectionner la variable désirée.

  3. Définissez les variables x1Min et x1Max avec le nom des variables de la Crosstable à partir desquelles vous voulez lire les valeurs utilisées pour représenter l'intervalle d'abscisses de la piste 1. S'il est fixe, mettez le nombre lui-même à la place du nom de la variable sinon appuyez sur le bouton “…” pour sélectionner la variable désirée. Pour donner un effet à un graphique déroulant, laisser les variables par défaut : PLC_timeMin et PLC_timeMax.

  4. Définir la variable y1Variable avec le nom de la variable de la Crosstable à partir de laquelle vous voulez lire la valeur utilisée pour représenter l'ordonnée de la piste 1. Appuyez sur le bouton “…” pour sélectionner la variable désirée.

  5. Définissez les variables y1Min et y1Max avec le nom des variables de la Crosstable à partir desquelles vous voulez lire les valeurs utilisées pour représenter l'intervalle d'ordonnée de la piste 1. S'il est fixe, mettez le nombre lui-même à la place du nom de la variable sinon appuyez sur le bouton “…” pour sélectionner la variable désirée. Définissez la variable x2Variable avec le nom de la variable de la table croisée à partir de laquelle vous voulez lire la valeur utilisée pour représenter l'abscisse de la piste 2. Appuyez sur le bouton “…” pour sélectionner la variable désirée.

  6. Définissez les variables x2Min et x2Max avec le nom des variables de la Crosstable à partir desquelles vous voulez lire les valeurs utilisées pour représenter l'intervalle d'abscisses de la piste 2. S'il est fixe, mettez le nombre lui-même à la place du nom de la variable sinon appuyez sur le bouton “…” pour sélectionner la variable désirée. Pour donner un effet à un graphique déroulant, laisser les variables par défaut : PLC_timeMin et PLC_timeMax.

  7. Définir la variable y2Variable avec le nom de la variable de la Crosstable à partir de laquelle vous voulez lire la valeur utilisée pour représenter l'ordonnée de la piste 2. Appuyez sur le bouton “…” pour sélectionner la variable désirée.

  8. Définissez les variables y2Min et y2Max avec le nom des variables de la Crosstable à partir desquelles vous voulez lire les valeurs utilisées pour représenter l'intervalle d'ordonnée de la piste 2. S'il est fixe, mettez le nombre lui-même à la place du nom de la variable sinon appuyez sur le bouton “…” pour sélectionner la variable désirée.

  9. Définissez les variables Display1 et Display2 avec le nom des variables de la Crosstable dont vous voulez lire le déclencheur pour afficher respectivement la piste 1 et la piste 2. S'ils sont vides, les déclencheurs sont automatiquement activés lorsque l'une des valeurs d'entrée change. Appuyez sur le bouton “…” pour sélectionner la variable désirée.
