Systempasswort ändern

Nicht verwendbar seit Version 3.3.x.

Standardmäßig sind das System “Username” und “Password”:

  • Username: root

  • Password: root

Um das Passwort zu ändern, folgen Sie der Reihenfolge der Anweisungen, die in ssh / telnet zu schreiben sind:

  • mount -o remount,rw /

  • passwd

  • mount -o remount,ro /

Befolgen Sie die folgenden Schritte im Detail:

  • Öffnen Sie die “Eingabeaufforderung”

  • Schreiben Sie: telnet 192.168.5.211 (Geräte-IP-Adresse)

  • In “mect login:” Schreiben Sie: root

  • In “Password” Schreiben Sie: root

  • Schreiben Sie: mount -o remount,rw /

  • Drücken Sie “Enter”

  • Schreiben Sie: passwd

  • Drücken Sie “Enter”

  • Geben Sie Ihr Passwort ein und drücken Sie “Enter”

  • Bestätigen Sie das neue Passwort und drücken Sie “Enter”

  • Schreiben Sie: mount -o remount,ro /

  • Drücken Sie “Enter”

Das Systempasswort wurde nun geändert.

Um die HMI-Anwendung zu übertragen, müssen Sie das dem Programm zugeordnete Passwort im QtCreator ändern. Folgen Sie der Abbildung unten.
