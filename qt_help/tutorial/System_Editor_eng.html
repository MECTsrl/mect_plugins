System Editor

System Editor changes some configuration for HMI and communication parameters between panel and remote devices.

To start the “Crosstable Editor” see following figure:


Here the “System Editor”:

On top of the “System” tab there is the key to save the setup. Under this there are some tabs: System, SERIAL PORT 0hellip; that can be changed or not depending on ports available on the product.

System Editor” characteristics:

  • System: setup of all general features of the system:

    • Pwd Logout Page: page shown at the end of session. If empty the actual page is shown.

    • Pwd Timeout [s]: timeout in seconds for the session. If “0” there is no timeout.

    • Screen Saver [s]: timeout in seconds for “Screen Saver”. If “0” the screen save is disabled.

    • Slow Log Period [s]: interval time for storage of “S” type variables (defined in the field “Update” of “Crosstable Editor”).

    • Fast Log Period [s]: interval time for storage of “F” type variables (defined in the field “Update” of “Crosstable Editor”). Number must be less than field “Slow Log Period”.

    • Max Log Space [Mb]: space to store the history variables.

    • Trend Window [s]: time value in seconds for the trend window. Must be at least 3 times of field “Fast Log Period”.

    • Retries: number of allowed errors for a device before slowing the query time.

    • Blacklist: reduced rate of interrogation in the case where it exceeds the “Retries”.

    • Start Page: name of the page to show first. In Qt Creator there must be a page like this. Only the pages present when the project is opened are displayed.

    • Home Page: page name for the home page. In Qt Creator there must be a page like this. Only the pages present when the project is opened are displayed.

    • Buzzer Touch: enable a buzzer for the touch screen touch.

    • Buzzer Alarm: enable a buzzer on alarm rising.

    • Read Period 1 [ms]: sets reading interval time for variables. Associated to “1” element in the field “Priority” of the Crosstable.

    • Read Period 2 [ms]: sets reading interval time for variables. Associated to “2” element in the field “Priority” of the Crosstable.

    • Read Period 3 [ms]: sets reading interval time for variables. Associated to “3” element in the field “Priority” of the Crosstable.

      The “Priority” field is not considered for variables with PLC protocol.

  • SERIAL PORT 0 / 1 / 2 / 3: setup of all serial communication parameter of the device.

    In the “Max Block Size” field the maximum number of variables inside a block is defined.

    WARNING: blocks reorder takes place after Crosstable save. Parameters of internal serial line must not be changed, otherwise panel doesn’t work.

  • TCP IP PORT: setup of TCP IP communication parameters with a remote device:

    • Timeout [ms]: timeout for the master to wait for slave answer.

    • Silence [ms]: frequency of consecutive request of a master to a slave.

    • Max Block Size [1…64]: define the maximum number of variables inside a block. WARNING: blocks reorder takes place after Crosstable save.

  • CANOPEN 0 / 1: setup of Baudrate of CANopen protocol.
