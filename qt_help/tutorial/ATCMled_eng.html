ATCMled

Object to display an integer Crosstable variable like a led: if zero the led is off, otherwise the led is on.

To use follows the steps:

  1. Drag object on page.

  2. Double click on object.

  3. Define variable field with Crosstable variable name to read. Press “…” button to select desired variable.

  4. Define visibilityVar variable with Crosstable variable to read value to show or not the object. Press “…” button to select desired variable.
