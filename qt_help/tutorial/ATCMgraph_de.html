ATCMgraph

Dieses Objekt ermöglicht die Anzeige eines zweispurigen Graphen.

Um es zu verwenden, führen Sie die folgenden Schritte aus:

  1. Ziehen Sie das Objekt auf die Seite und doppelklicken Sie.

  2. Definieren Sie die Variable x1Variable mit dem Namen der Crosstable-Variablen, aus dem Sie den Wert lesen möchten, der zur Darstellung der Abszisse von Spur 1 verwendet wird. Drücken Sie die Taste “…”, um die gewünschte Variable auszuwählen.

  3. Definieren Sie die Variablen x1Min und x1Max mit dem Namen der Crosstable-Variablen, aus dem Sie die Werte lesen möchten, die zur Darstellung des Abszissenintervalls von Spur 1 verwendet werden. Wenn es fest ist, geben Sie die Zahl selbst anstelle des Variablennamens ein, andernfalls drücken Sie die Taste “…”, um die gewünschte Variable auszuwählen. Um einem scrollenden Diagramm einen Effekt zu verleihen, lassen Sie die Standardvariablen: PLC_timeMin und PLC_timeMax.

  4. Definieren Sie die Variable y1Variable mit dem Namen der Crosstable-Variablen, aus dem Sie den Wert lesen möchten, der für die Ordinate von Spur 1 verwendet wird. Drücken Sie die Taste “…”, um die gewünschte Variable auszuwählen.

  5. Definieren Sie die Variablen y1Min und y1Max mit dem Namen der Crosstable-Variablen, aus denen Sie die Werte für das Ordinatenintervall von Spur 1 lesen möchten. Wenn es fest ist, geben Sie die Zahl selbst anstelle des Variablennamens ein, andernfalls drücken Sie die Taste "....", um die gewünschte Variable auszuwählen. Definieren Sie die Variable x2Variable mit dem Namen der Variablen des Crosstable, aus dem Sie den Wert lesen möchten, der zur Darstellung der Abszisse von Spur 2 verwendet wird. Drücken Sie die Taste “…”, um die gewünschte Variable auszuwählen.

  6. Definieren Sie die Variablen x2Min und x2Max mit dem Namen der Crosstable-Variablen, aus dem Sie die Werte lesen möchten, die zur Darstellung des Abszissenintervalls von Spur 2 verwendet werden. Wenn es fest ist, geben Sie die Zahl selbst anstelle des Variablennamens ein, andernfalls drücken Sie die Taste “…”, um die gewünschte Variable auszuwählen. Um einem scrollenden Diagramm einen Effekt zu verleihen, lassen Sie die Standardvariablen: PLC_timeMin und PLC_timeMax..

  7. Definieren Sie die Variable y2Variable mit dem Namen der Crosstable-Variablen, aus dem Sie den Wert lesen möchten, der für die Ordinate von Spur 2 verwendet wird. Drücken Sie die Taste “…”, um die gewünschte Variable auszuwählen.

  8. Definieren Sie die Variablen y2Min und y2Max mit dem Namen der Crosstable-Variablen, aus denen Sie die Werte für das Ordinatenintervall von Spur 2 lesen möchten. Wenn es fest ist, geben Sie die Zahl selbst anstelle des Variablennamens ein, andernfalls drücken Sie die Taste “…”, um die gewünschte Variable auszuwählen.

  9. Definieren Sie die Variablen Display1 und Display2 mit dem Namen der Crosstable-Variablen, aus denen Sie den Trigger lesen möchten, um Track 1 bzw. Track 2 anzuzeigen. Wenn leer, werden die Trigger automatisch aktiviert, wenn sich einer der Eingangswerte ändert. Drücken Sie die Taste “…”, um die gewünschte Variable auszuwählen.
