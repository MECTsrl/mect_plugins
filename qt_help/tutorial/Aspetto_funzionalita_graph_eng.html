Look and functionality customization

Selecting the element, its property are shown in the lower right size of the window of QtCreator in the section ATCMgraph. Modifying one or more properties the graphic aspect and some functionalities can be modified. To modify them select right column of the corresponding row.

In particular for graphic aspect:

  • x1Step, y1Step, x2Step and y2Step: visualization step for values on traces 1 e 2.

  • x1TickLabelVisible, y1TickLabelVisible, x2TickLabelVisible and y2TickLabelVisible: axis visualization on off.

  • x1TickLabelFormat, y1TickLabelFormat, x2TickLabelFormat and y2TickLabelFormat: shows values on axis i numeric or hour format hh:mm:ss.

  • x1Label, y1Label, x2Label and y2Label: title of the axis.

  • pen1Color and pen2Color: color of each trace.

  • gridVisible: grid on off.

  • legendVisible: legend on off.

  • title: title on off.

  • bgColor: background color of the object.

  • refresh: refresh time of the object in milliseconds.

  • viewStatus: if checked the color is green or red if button is used as toggle, depending on setting of variable and if the edge is visible.

WARNING: in all Properties, except ATCM plugin, it is possible to see the “Context Help” of Qt to see a description of the functionality: select the desired row and press “F1” key.

To use the code properties, to write in Help →Index: QFrame and QwtPlotx.
