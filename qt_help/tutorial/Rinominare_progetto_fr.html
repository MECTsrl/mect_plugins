Renommer un projet

Pour renommer le dossier de projet créé avec QtCreator, vous devez suivre les étapes suivantes:

  1. Assurez-vous que vous avez fermé le projet que vous souhaitez renommer.

  2. Renommez le dossier du projet (n'insérez pas de caractères spéciaux (sauf _) ni d'espaces.

  3. Supprimez le fichier avec l'extension “.pro.user” du dossier du projet.

  4. Renommez le fichier avec l'extension “.pro” avec le nom que vous avez choisi à l'étape 2.

  5. Renommez le fichier avec l'extension “.pro.shared” avec le nom que vous avez choisi à l'étape 2.

  6. Ouvrez le nouveau projet depuis QtCreator.

  7. Lorsque l'image suivante apparaît, appuyez sur le bouton “Configure Project”.

  1. Effectuez un “Run qmake” et un “Clean All”, voyez le chapitre “Erreurs de compilation”

  2. Faites le “Build” du projet.

  3. Définissez les paramètres d'envoi du projet à l'appareil:

    • Désactivez, sous la rubrique “Upload files via SFTP”, le flag “Incremental deployment”;

    • Cliquez sur “Add Deploy Step” et sélectionnez “Run custom remote command”

    • Écrivez “stop” (en minuscule) et déplacez le bloc “Run custom remote command” d'une position vers le haut, à l'aide de la flèche surlignée en jaune (appuyez sur Details).
