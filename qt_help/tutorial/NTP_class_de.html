NTP Client Klasse

Die Klasse ntpclient ist eine Bibliothek zur Synchronisation mit dem NTP-Protokoll, die in Ihre Projekte eingebunden werden kann.

Die Funktionen, die verwendet werden können, sind:
  1. QString getNtpServer();
  2. int getTimeout_s();
  3. int getOffset_h();
  4. int getPeriod_h();
  5. void setNtpParams(const QString &ntpServer, int ntpTimeout_s, int ntpOffset_h, int ntpPeriod_h);
  6. void requestNTPSync();
  7. void requestDateTimeChange(QDateTime newTime);

Im Beispielprojekt time_set_AnyTPAC043 in mect_apps/Examples werden alle diese Funktionen verwendet, um die Systemseite Time Set neu zu erstellen.
