Transferencia de la aplicación al dispositivo

Para poder transferir su aplicación del PC al panel , debe conectar el panel al PC a través de un cable de red o comprobar que ambos están conectados entre sí.

La primera vez que quiera enviar el ejecutable (Deploy) al panel, seleccione “Projects” → “Run” y escriba “stop” (todos en minúsculas) en “Run custom remote command” :

Es aconsejable trabajar en modo “Release” porque la aplicación es menos “pesante” para el objetivo.

Si es necesario, puede especificar una dirección IP para el objetivo en “Tools → Options → Devices”.

Es aconsejable realizar una prueba para comprobar que existe una conexión entre el PC y el panel operador; para ello, pulse el botón “Test”.

La frase de paso es: Vpn:Update1207

Se utiliza una llave común para todos los dispositivos Mect situada en C:\Qt485\keys para desplegar o acceder al dispositivo.

Se recomienda encarecidamente generar y utilizar una nueva llave para sus proprios dispositivos (Putty Installer: https://www.chiark.greenend.org.uk/~sgtatham/putty/latest.html).
