Activer / désactiver un noeud Modbus RTU-TCP

Il est possibile d'activer ou désactiver un nœud Modbus RTU / TCP via l'utilisation de la variable de diagnostic “NODE_xx_STATUS”

Le nœud doit être activé ou désactivé par l'IHM.

  • Nœud désactivé: doWrite_NODE_xx_STATUS(0);

  • Nœud activé: doWrite_NODE_xx_STATUS(1);

Il est possible de connaître le numéro de noeud via l'onglet “Devices”.
