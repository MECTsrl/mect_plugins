Password operating principle

There are 2 different ways to view / set passwords:

  • Use the dedicated “set_password” library page.

  • Use the generic “menu” library page.

Using the “set_password” page:

To view the password setting page, use an ATCMbutton and enter the name of the library page “set_password” in the Property “pageName”.

It is advisable to protect the page change of the ATCMbutton with a different password, static or dynamic, by inserting it in the “PasswordVar” field.

In the displayed page, it is possible to set the passwords of the 3 levels: “Admin”, “SuperUser” and “User”.


Using the “menu” page:

All systems have default password equal to “0” for all 3 levels.

As default passwords are the same for the 3 levels, first login has highest priority. So first login is assigned to “Admin” and the ‘Password’ key modifies the “Admin” password.

Next login with “0” password is assigned to “SuperUser” and it is possible to modify its password. Next login with “0” password is assigned to “User” and it is possible to modify its password.

Last login with “0” password is assigned to “Operator”.
