Device variables (Local I/O)

Inside the Crosstable there are the most important product variables. Specifically, they are variable from the 5300 line of the “Crosstable Editor”.

For example, if in HMI program you want to command an ATCMbutton to change the digital output 1 or to initialize in PLC digital output 1 to a value, use the variable PLC_DigOut_1.

See the “TPAC1008_02 models” section to view the list of variables on the TPAC1008_02 models.

See the “TPAC1007 models” section to view the list of variables on the TPAC1007 models.

See the “TPAC1008_03 model” section to view the list of variables on the TPAC1008_03 model.

See the “TPLC050 / TPLC100 models” section to view the list of variables on the TPLC050 and TPLC100 models.
