Festlegen von Passwörtern

Es gibt 2 verschiedene Möglichkeiten, Passwörter anzuzeigen / festzulegen:

  • Verwenden Sie die dedizierte “set_password”-Bibliotheksseite.

  • Verwenden Sie die generische “menu”.

Verwenden der Seite “set_password”:

Um die Passwort-Setup-Seite anzuzeigen, verwenden Sie eine ATCMbutton und geben Sie den Namen der Bibliotheksseite “set_password” in die Eigenschaft “pageName” ein.

Es ist ratsam, den Seitenwechsel des ATCMbutton mit einem anderen Passwort, statisch oder dynamisch, zu schützen, indem Sie es in das Feld “PasswordVar” einfügen.

Auf der angezeigten Seite können die Passwörter der 3 Ebenen festgelegt werden: “Admin”, “SuperUser” und “User”.


Verwenden der Seite “menu”:

Alle Systeme haben ein Standardpasswort von “0” für alle drei Ebenen.

Da die Standardpasswörter für alle drei Ebenen identisch sind, wird der erste Anmeldevorgang, den die Anwendung durchführt, hierarchisch von der höchsten Prioritätsstufe aus verwaltet. Daher wird der erste Anmeldung, der in der Anwendung durchgeführt wird, “Admin” zugewiesen und jeder Druck auf die Taste ‘Password’ ändert das Passwort von “Admin”.

Die nächste Anmeldung mit dem Passwort “0” wird dem Benutzer “SuperUser” zugeordnet, dessen Passwort auf Wunsch geändert werden kann. Die nächste Anmeldung mit dem Passwort “0” wird dem Benutzer “User” zugeordnet, dessen Passwort auf Wunsch geändert werden kann.

Die letzte Anmeldung mit dem Passwort “0” wird dem “Operator” zugeordnet.
