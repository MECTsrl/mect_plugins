Funcionalidad Editor de Crosstable

  • Goto Section: permite la navegación dentro de la Crosstable.

    • Retentive [1 - 192]: variables que mantienen su valor al apagar el dispositivo;

    • Not Retentive [193 - 4999]: variables que pierden su valor al apagar el dispositivo;

    • Diagnostic [5000 - 5171]: variables de diagnóstico. Se utilizan cuando se comunica con un protocolo distinto a “PLC”. Para más información ver el capítulo “Variables de diagnóstico”

    • Node Status [5172 - 5299]: variables de estado de nodo. Se utilizan cuando se comunica con un protocolo distinto a “PLC”.

    • Local I/O [5300 - 5389]: variables de entrada y salida del dispositivo. Estas variables pueden ser renombradas y modificadas (en algunas partes) para permitir una mayor flexibilidad en la escritura de la aplicación. Para más información lea el capítulo “Variables de producto”;

    • System [5390 - 5472]: variables del sistema. Para más información lea el capítulo “Variables de producto”.

  • Permite expandir o comprimir las líneas vacías de la Crosstable.

  • Realice una búsqueda avanzada de una o más variables dentro la Crosstable.

  • Permite cargar el archivo “Crosstable.csv” presente en otro proyecto. El archivo importado sobrescribirá el archivo actual en el momento de guardarlo.

  • Habilite la edición múltiple de variables dentro la Crosstable.

  • Permite cancelar la última acción realizada y volver a la configuración anterior.

  • Permite guardar los ajustes actuales. Si hace un cambio, el borde del botón estará en rojo, de lo contrario estará en verde.

  • Verifica la exactitud de la Crosstable.

  • Inicia el programa de PLC. Si no hay errores en la Crosstable, puede abrir el software del PLC. Para más información ver el capítulo “Abrir el software PLC”.

  • Habilitar todas y sólo las variables de diagnóstico de los nodos y buses de campo existentes (véase la sección crosstable Diagnostic y Node Status).

Pulsando el botón derecho del ratón sobre la tabla se pueden realizar algunas operaciones:

  • Goto Line: selecciona directamente la línea deseada.

  • Insert Blank Rows: Insertar una línea vacía.

  • Clear Rows: Borra el contenido de la línea seleccionada.

  • Delete Rows: Borra una o más líneas seleccionadas.

  • Copy Rows: copia una o más líneas seleccionadas.

  • Cut Rows: : corta una o más líneas seleccionadas.

  • Paste Rows: Pega una o más líneas seleccionadas.

  • Paste MPNC006 Modules: Pegar todas las variables asociadas a los modelos remotos MPNC005 y MPNC006 MPNC006 y módulos conectados (MPNC020 / MPNC030 / MPNC035).

  • Paste MPNE1001 Module: Pegar todas las variables asociadas a los modelos remotos MPNE10.

  • Paste TPLC050 Modules: Pegar todas las variables asociadas a los módulos remotos (MPNC020 / MPNC030 / MPNC035).

  • Paste MPSN100 Module: Pegar todas las variables asociadas a los sensores de la familia MPSN100 (temperatura, humedad...).


Variables de ALARMA / EVENTO:

Si queremos utilizar una variable como alarma/evento debemos indicarlo en el campo “Behavior” pero sólo es posible si:

  • El campo “Priority” es diferente de “0”;

  • El campo “Update” es diferente de “H”;

  • El campo “Type” es igual a “BIT”;

Con estos ajustes puede elegir la condición de alarma/evento.

Las variables presentes en el menú desplegable son todas las presentes en la Crosstable, incluidas las del sistema, a excepción de las variables que tienen en el campo “Update” el valor “H” y las que tienen en el campo “Priority” el valor “0”.
