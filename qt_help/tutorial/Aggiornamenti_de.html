Aktualisierung

Lesen Sie vor der Aktualisierung die Datei "changelog.txt" auf der Mect-Website im Bereich “DOWNLOAD Mect Suite”.

Laden Sie zum Aktualisieren die Dateien von der mect.it-Website herunter, insbesondere: Software and Services => Mect Suite => DOWNLOAD Mect Suite => Stable release 3.x.x => Mect Suite 3.x.x => Update Device

ACHTUNG:SCHALTEN SIE DAS SYSTEM WÄHREND DER AKTUALISIERUNG NICHT AUS.

Aktualisierung des Geräts:

  • img_sysupdate-3.x-ProduktName.ext2 und sysupdate_img_3.x.sh:

    • Via USB: kopieren Sie die Dateien “img_sysupdate-3.x-ProduktName.ext2” und “sysupdate_img_3.x.sh” auf einem USB-Stick.

      Zum Beispiel müssen Sie für das TPAC1007_04_AC die Dateien auf dem Stick kopieren:

      • img_sysupdate-3.0.0-TPAC1007_04_AC.ext2

      • sysupdate_img_3.0.0.sh

    Für Version 3.x dauret es ca.2 Minuten. Nach dem Anzeigen der Nachricht: “Please power off and remove the USB storage device” Konnen Sie das Gerät ausschalten, den USB_Stick entfernen und neu starten. Bei den Modellen TPAC1007 und TP1043 wird der Micro-USB-Adapter(Typ A) - USB benötigt.

Geräte ohne Monitor: warten Sie ca 4 Minuten, bevor Sie das Gerät ausschalten.

ACHTUNG: Ab Version 3.0.0 ist nicht möglich, über Browser zu aktualisieren.
